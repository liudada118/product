## 1. 使用说明

| 项目 | 内容 |
|---|---|
| 用例编号 | 按模块编号，如 `TC-CON-001` |
| 场景类型 | 正向、异常、边界、数据口径、性能、兼容 |
| 优先级 | P0 必测，P1 建议本轮覆盖，P2 后续覆盖 |

| 状态 | 默认为未执行，可在飞书中改为未执行、通过、失败、阻塞、待确认 |
| 负责人 | 复制到飞书后由 QA 或研发补充 |

## 2. 测试环境和通用前置条件

| 项目 | 要求 |
|---|---|
| 软件版本 | JQ Tools V2.0 待测版本 |
| 操作系统 | Windows，macOS 如本轮覆盖则单独记录 |
| 测试设备 | 至少一套合法汽车坐垫设备、一套合法汽车靠背设备 |
| 异常设备 | 至少准备一个未配置 MAC 或未授权设备 |

| 数据准备 | 至少准备一条可回放历史数据、一份合法 CSV、一份非法 CSV |
| 测试原则 | P0 用例必须真实设备验证，不能只用模拟数据 |
