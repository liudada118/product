## 2. 画布翻转业务逻辑

画布翻转/旋转用于让用户把传感器实际摆放方向和软件分析方向对齐。当前产品口径确认：翻转/旋转不是单纯显示效果，而是方向修正能力。方向修正后，后续展示、统计、采集、回放、导出都应基于修正后的矩阵。

说明：底层串口原始帧 rawFrame 可保留用于排查；但业务上的 rawData / storedData / exportedData 应按用户确认后的方向保存和使用。

| 项目 | 业务规则 | 优先级 |
|---|---|---|
| 触发入口 | 用户点击左右翻转或上下翻转按钮 | P0 |
| 旋转入口 | 用户点击 90 度旋转按钮 | P0 |
| 影响对象 | 当前对象的业务矩阵方向，坐垫和靠背分开处理 | P0 |
| 左右翻转 | 当前矩阵按横向镜像修正 | P0 |
| 上下翻转 | 当前矩阵按纵向镜像修正 | P0 |
| 90 度旋转 | 当前矩阵按顺时针或逆时针 90 度旋转，方向需明确 | P0 |
| 双向翻转 | 同时开启左右和上下翻转时，相当于旋转 180 度 | P0 |
| 坐垫/靠背独立 | 座椅坐垫和靠背分别保存方向状态，互不影响 | P0 |
| 是否影响 rawFrame | 不影响，保留设备原始帧用于排查 | P0 |
| 是否影响 rawData | 影响。业务 rawData 应按修正后的方向生成和保存 | P0 |
| 是否影响 displayData | 影响。展示基于修正后的矩阵 | P0 |
| 是否影响 calculatedData | 影响。统计、坐标、重心基于修正后的矩阵 | P0 |
| 是否影响采集 | 影响。采集保存修正方向后的全量矩阵，并记录方向配置 | P0 |
| 是否影响回放 | 影响。回放直接使用已持久化的修正方向矩阵 | P0 |
| 是否影响 CSV | 影响。CSV 导出修正方向后的矩阵，并导出方向配置字段 | P0 |
| 是否持久化 | 是。方向配置需要按设备类型 + 显示对象保存到本地 | P0 |

## 6. 量尺业务逻辑

量尺用于测量两个传感点或一个区域之间的真实距离。量尺是查看和分析工具，不影响原始数据、采集数据和统计数据。

| 项目 | 业务规则 | 优先级 |
|---|---|---|
| 使用条件 | 仅 2D 数字视图下可用 | P0 |
| 触发入口 | 用户点击量尺按钮进入测量模式 | P1 |
| 互斥规则 | 量尺和框选不能同时启用 | P0 |
| 测量对象 | 两个传感点之间的距离，或矩形区域的宽高距离 | P1 |
| 计算依据 | `pointWidthDistance` 和 `pointHeightDistance` | P0 |
| 默认单位 | 推荐使用 cm，必要时展示 mm | P1 |
| 缩放影响 | 缩放只影响显示，不影响真实距离 | P0 |
| 翻转影响 | 翻转只改变显示位置，不改变真实距离 | P0 |
| 采集影响 | 不影响采集，不入库为采集数据 | P0 |
| 回放影响 | 回放中可使用量尺测量历史画面 | P1 |
| 导出影响 | V2.0 不导出量尺结果，后续可支持截图/报告 | P2 |

## 7. 量尺主流程

| 步骤 | 用户动作 | 系统处理 | 成功结果 | 异常处理 |
|---|---|---|---|---|
| 1 | 点击量尺按钮 | 进入量尺模式，鼠标指针变为测量状态 | 可开始测量 | 非 2D 数字视图提示不可用 |
| 2 | 点击或拖拽起点 | 记录起点传感点坐标 | 显示起点 | 起点超出有效矩阵时不记录 |
| 3 | 点击或拖拽终点 | 记录终点传感点坐标 | 显示测量线和距离 | 终点超出有效矩阵时提示 |
| 4 | 查看测量结果 | 根据点距计算横向、纵向、直线距离 | 展示距离数值 | 点距缺失时提示无法计算 |
| 5 | 新增多条量尺 | 保存当前量尺并继续测量 | 画布显示多条量尺 | 超过上限时提示 |
| 6 | 选中量尺 | 高亮当前量尺 | 可删除或查看详情 | 无 |
| 7 | 删除量尺 | 删除当前量尺 | 画布更新 | 删除失败时保持原状 |
| 8 | 退出量尺 | 关闭量尺模式 | 鼠标恢复默认状态 | 当前量尺是否保留按方案处理 |

## 7.1 量尺拖拽与端点移动实现细节

本节补充两类编辑能力：拖拽整条量尺、移动单个端点。两者都只修改量尺对象本身，不影响 rawData、displayData、calculatedData、采集数据和历史数据。

### 7.1.1 交互对象和优先级

| 交互区域 | 触发条件 | 操作含义 | 优先级 |
|---|---|---|---|
| 起点端点 | 鼠标 hover 起点圆点或触控命中起点热区 | 移动起点，终点不动 | P0 |
| 终点端点 | 鼠标 hover 终点圆点或触控命中终点热区 | 移动终点，起点不动 | P0 |
| 量尺线段 | 鼠标 hover 线段主体或距离标签 | 拖拽整条量尺，起点和终点一起移动 | P0 |
| 空白画布 | 未命中端点或线段 | 新建量尺或取消选中 | P0 |

命中优先级：端点高于线段，线段高于画布。端点和线段热区可以大于视觉图形，避免用户难以选中。

### 7.1.2 推荐热区和光标

| 元素 | 视觉尺寸 | 推荐命中热区 | 光标/反馈 |
|---|---|---|---|
| 起点/终点 | 6-8 px 圆点 | 12-16 px | 十字或拖拽光标 |
| 线段主体 | 1-2 px 线条 | 8-12 px | 移动光标 |
| 距离标签 | 按文本尺寸 | 标签外扩 4 px | 移动光标 |

选中状态下，端点显示实心圆点，线段加粗或变亮，右侧量尺列表同步高亮当前量尺。

### 7.1.3 拖拽整条量尺

拖拽整条量尺用于保持测量长度和方向不变，只改变量尺在矩阵中的位置。

| 步骤 | 用户动作 | 系统处理 | 结果 |
|---|---|---|---|
| 1 | 鼠标移动到量尺线段或距离标签 | 线段高亮，光标变为移动状态 | 用户知道可拖拽 |
| 2 | 按下鼠标左键 | 记录拖拽开始时的起点、终点和鼠标所在矩阵坐标 | 进入 dragging-ruler 状态 |
| 3 | 拖动鼠标 | 将鼠标移动距离换算成 row/column 偏移量 | 预览新位置 |
| 4 | 实时校验边界 | 起点和终点都必须落在有效矩阵范围内 | 不允许拖出有效区域 |
| 5 | 松开鼠标 | 写入新的起点和终点坐标，重新计算距离 | 量尺完成移动 |

计算规则：

| 项目 | 规则 |
|---|---|
| 偏移量 | `deltaRow = currentRow - dragStartRow`，`deltaColumn = currentColumn - dragStartColumn` |
| 新起点 | `newStart = oldStart + delta` |
| 新终点 | `newEnd = oldEnd + delta` |
| 吸附规则 | 拖拽过程中按传感点 row/column 吸附，最终坐标为整数传感点坐标 |
| 距离结果 | 整条量尺拖拽不改变横向距离、纵向距离和直线距离 |
| 边界处理 | 如果新起点或新终点越界，使用最大可用 delta 进行钳制，保证整条量尺仍在有效区域内 |
| 不规则有效区 | 如靠背存在 mask，有效性以端点所在传感点是否有效为准 |

说明：整条量尺拖拽只移动位置，不改变长度。如果拖到边界，系统应停止在最后一个合法位置，而不是让量尺变短。

### 7.1.4 移动单个端点

移动端点用于修正用户点错的位置，或调整测量长度。

| 步骤 | 用户动作 | 系统处理 | 结果 |
|---|---|---|---|
| 1 | 鼠标移动到起点或终点 | 当前端点放大或高亮 | 用户知道可编辑 |
| 2 | 按下鼠标左键 | 记录固定端点和被拖动端点 | 进入 dragging-endpoint 状态 |
| 3 | 拖动端点 | 被拖动端点吸附到最近有效传感点 | 预览新线段 |
| 4 | 实时计算 | 更新横向距离、纵向距离、直线距离和标签 | 用户实时看到结果 |
| 5 | 松开鼠标 | 保存新端点坐标 | 完成端点修改 |

计算规则：

| 项目 | 规则 |
|---|---|
| 固定端点 | 拖动起点时终点不动；拖动终点时起点不动 |
| 被拖动端点 | 始终吸附到最近的有效传感点中心 |
| 距离结果 | 根据新端点和固定端点重新计算 |
| 起终点相同 | 允许临时预览 0；松开时提示“测量距离为 0”，建议用户重新选择 |
| 越界处理 | 被拖动端点越界时不更新坐标，保持最后一个合法坐标 |
| mask 无效点 | 被拖动端点落在无效点时不更新坐标，并提示“请在有效区域内测量” |

### 7.1.5 状态机建议

| 状态 | 进入条件 | 可响应操作 | 退出条件 |
|---|---|---|---|
| idle | 未开启量尺或未选中量尺 | 进入量尺模式、选中量尺 | 点击量尺按钮 |
| creating-start | 量尺模式下等待起点 | 点击有效传感点 | 起点创建成功 |
| creating-end | 起点已选，等待终点 | 移动预览、点击终点 | 终点创建成功或取消 |
| selected | 已选中一条量尺 | 拖拽线段、拖拽端点、删除、重命名 | 取消选中或删除 |
| dragging-ruler | 在线段/标签上按下鼠标 | 移动整条量尺 | 松开鼠标或按 Esc |
| dragging-endpoint | 在端点上按下鼠标 | 移动单个端点 | 松开鼠标或按 Esc |

Esc 规则：拖拽中按 Esc 取消本次拖拽，恢复拖拽前坐标；创建中按 Esc 取消当前未完成量尺。

### 7.1.6 与缩放、翻转、旋转关系

| 场景 | 规则 |
|---|---|
| 缩放后拖拽 | 鼠标命中基于屏幕坐标，但最终计算必须转换为矩阵 row/column |
| 左右/上下翻转后拖拽 | 用户看到的显示坐标转换为修正后的矩阵坐标，再保存量尺坐标 |
| 90 度旋转后拖拽 | 使用旋转后的矩阵宽高和当前 displayType 点距配置 |
| 坐垫/靠背切换 | 当前对象的量尺只在当前对象内可编辑，不跨对象拖拽 |
| 整体视图 | 如整体视图包含坐垫和靠背，量尺编辑应限定在当前命中的对象内 |

### 7.1.7 异常和保护规则

| 异常 | 处理 | 用户提示 |
|---|---|---|
| 拖拽时设备断开 | 保留量尺编辑状态，但不影响连接状态 | 设备已断开，量尺仅用于当前画面查看 |
| 拖拽中切换视图 | 取消本次拖拽，恢复拖拽前坐标 | 视图已切换，本次拖拽已取消 |
| 拖拽中开始采集 | 允许完成量尺编辑，但量尺不进入采集数据 | 无需提示 |
| 拖拽到无效区域 | 坐标不更新，保持最后合法位置 | 请在有效区域内测量 |
| 当前量尺被锁定 | 不允许拖拽线段或端点 | 当前量尺已锁定 |
| 当前量尺隐藏 | 不参与命中检测 | 无 |

| 编号 | 验收项 | 通过标准 | 优先级 |
|---|---|---|---|
| 验收-量尺拖拽-01 | 整条拖拽 | 拖拽线段后起点和终点同步移动，距离不变 | P0 |
| 验收-量尺拖拽-02 | 端点移动 | 拖动起点或终点后，另一端固定，距离实时更新 | P0 |
| 验收-量尺拖拽-03 | 吸附传感点 | 拖拽后端点坐标均为有效传感点 row/column | P0 |
| 验收-量尺拖拽-04 | 越界保护 | 拖出有效区域时量尺停在最后合法位置 | P0 |
| 验收-量尺拖拽-05 | 缩放稳定 | 缩放到不同倍率后拖拽，真实距离计算正确 | P0 |
| 验收-量尺拖拽-06 | 翻转稳定 | 翻转后拖拽，端点位置和真实距离正确 | P0 |
| 验收-量尺拖拽-07 | 旋转稳定 | 90 度旋转后拖拽，矩阵坐标和距离正确 | P0 |
| 验收-量尺拖拽-08 | 取消拖拽 | 拖拽中按 Esc 恢复拖拽前坐标 | P1 |

## 8. 量尺计算规则

| 项目 | 公式/规则 | 优先级 |
|---|---|---|
| 点坐标 | 使用矩阵坐标，不使用 DOM 像素坐标 | P0 |
| 横向距离 | abs(x2 - x1) x pointWidthDistance | P0 |
| 纵向距离 | abs(y2 - y1) x pointHeightDistance | P0 |
| 直线距离 | sqrt(横向距离² + 纵向距离²) | P1 |
| 单位换算 | mm 转 cm 时除以 10 | P1 |
| 缩放后计算 | 仍按矩阵坐标和点距计算，不按屏幕像素计算 | P0 |
| 翻转后计算 | 翻转前后同两点真实距离一致 | P0 |
| 坐垫/靠背切换 | 使用当前 displayType 对应的点距配置 | P0 |
| 不同设备 | 使用当前设备类型对应点距配置 | P0 |

## 11. 待确认项和方案

| 量尺形态 | 两点直线量尺 | 矩形宽高量尺 | 两者都支持 | V2.0 方案 A，P1 做方案 C |
| 多量尺数量上限 | 最多 4 条 | 最多 8 条 | 不限制 | 方案 B |
| 退出量尺后是否保留 | 当前会话保留 | 退出即清空 | 弹窗询问 | 方案 A |
| 回放中是否可用量尺 | 可用，只影响查看 | 禁用 | 只暂停时可用 | 方案 A |
| 量尺是否导出 | 不导出 | 导出到报告 | 导出到 CSV | V2.0 不导出，P2 做方案 B |
