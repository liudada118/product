## 2. 画布翻转业务逻辑

画布翻转/旋转用于让用户把传感器实际摆放方向和软件分析方向对齐。当前产品口径确认：翻转/旋转不是单纯显示效果，而是方向修正能力。方向修正后，后续展示、统计、采集、回放、导出都应基于修正后的矩阵。

说明：底层串口原始帧 rawFrame 可保留用于排查；但业务上的 rawData / storedData / exportedData 应按用户确认后的方向保存和使用。

| 项目 | 业务规则 | 优先级 |
|---|---|---|
| 触发入口 | 用户点击左右翻转或上下翻转按钮 | P0 |
| 旋转入口 | 用户点击 90 度旋转按钮 | P0 |
| 影响对象 | 当前对象的业务矩阵方向，坐垫和靠背分开处理 | P0 |
| 左右翻转 | 当前矩阵按横向镜像修正 | P0 |
| 上下翻转 | 当前矩阵按纵向镜像修正 | P0 |
| 90 度旋转 | 当前矩阵按顺时针或逆时针 90 度旋转，方向需明确 | P0 |
| 双向翻转 | 同时开启左右和上下翻转时，相当于旋转 180 度 | P0 |
| 坐垫/靠背独立 | 座椅坐垫和靠背分别保存方向状态，互不影响 | P0 |
| 是否影响 rawFrame | 不影响，保留设备原始帧用于排查 | P0 |
| 是否影响 rawData | 影响。业务 rawData 应按修正后的方向生成和保存 | P0 |
| 是否影响 displayData | 影响。展示基于修正后的矩阵 | P0 |
| 是否影响 calculatedData | 影响。统计、坐标、重心基于修正后的矩阵 | P0 |
| 是否影响采集 | 影响。采集保存修正方向后的全量矩阵，并记录方向配置 | P0 |
| 是否影响回放 | 影响。回放直接使用已持久化的修正方向矩阵 | P0 |
| 是否影响 CSV | 影响。CSV 导出修正方向后的矩阵，并导出方向配置字段 | P0 |
| 是否持久化 | 是。方向配置需要按设备类型 + 显示对象保存到本地 | P0 |

| 模块 | 影响规则 | 优先级 |

| 2D 热力图 | 热力图按修正后的方向显示 | P0 |
| 3D 点图 | 点位高度和颜色按修正后的矩阵映射 | P1 |
| 3D 模型 | 3D 模型本身不跟随翻转，不镜像模型；压力点是否按修正矩阵贴合需研发确认 | P1 |

## 5. 翻转异常处理

| 异常类型 | 判断条件 | 系统处理 | 用户提示 | 优先级 |
|---|---|---|---|---|
| 无数据翻转 | 当前无有效矩阵 | 可切换状态，但显示为空 | 暂无压力数据 | P1 |
| 采集中翻转 | 正在采集时点击翻转/旋转 | 禁止修改方向 | 正在采集，不能修改方向，请停止采集后再操作 | P0 |
| 回放中翻转 | 回放时点击翻转 | 仅影响当前查看，不修改历史数据 | 无 | P1 |
| 数据对比中方向不一致 | A/B 方向不同 | 进入对比前统一或提示 | 两组数据方向不同，请统一后对比 | P0 |
| 方向状态丢失 | 历史记录无 data_direction | 按 none 处理并提示 legacy 数据 | 历史数据缺少方向信息，已按默认方向显示 | P1 |
| 90 度旋转后矩阵尺寸变化 | 例如 50 x 64 旋转后变成 64 x 50 | 更新矩阵宽高并重新校验框选/量尺 | 当前方向已改变，相关区域已重新校验 | P0 |
| 持久化失败 | 本地配置写入失败 | 画面可临时生效，但提示保存失败 | 方向保存失败，下次打开可能不会保留 | P0 |

## 6. 量尺业务逻辑

量尺用于测量两个传感点或一个区域之间的真实距离。量尺是查看和分析工具，不影响原始数据、采集数据和统计数据。

| 项目 | 业务规则 | 优先级 |
|---|---|---|
| 使用条件 | 仅 2D 数字视图下可用 | P0 |
| 触发入口 | 用户点击量尺按钮进入测量模式 | P1 |
| 互斥规则 | 量尺和框选不能同时启用 | P0 |
| 测量对象 | 两个传感点之间的距离，或矩形区域的宽高距离 | P1 |
| 计算依据 | `pointWidthDistance` 和 `pointHeightDistance` | P0 |
| 默认单位 | 推荐使用 cm，必要时展示 mm | P1 |
| 缩放影响 | 缩放只影响显示，不影响真实距离 | P0 |
| 翻转影响 | 翻转只改变显示位置，不改变真实距离 | P0 |
| 采集影响 | 不影响采集，不入库为采集数据 | P0 |
| 回放影响 | 回放中可使用量尺测量历史画面 | P1 |
| 导出影响 | V2.0 不导出量尺结果，后续可支持截图/报告 | P2 |

## 7. 量尺主流程

| 步骤 | 用户动作 | 系统处理 | 成功结果 | 异常处理 |
|---|---|---|---|---|
| 1 | 点击量尺按钮 | 进入量尺模式，鼠标指针变为测量状态 | 可开始测量 | 非 2D 数字视图提示不可用 |
| 2 | 点击或拖拽起点 | 记录起点传感点坐标 | 显示起点 | 起点超出有效矩阵时不记录 |
| 3 | 点击或拖拽终点 | 记录终点传感点坐标 | 显示测量线和距离 | 终点超出有效矩阵时提示 |
| 4 | 查看测量结果 | 根据点距计算横向、纵向、直线距离 | 展示距离数值 | 点距缺失时提示无法计算 |
| 5 | 新增多条量尺 | 保存当前量尺并继续测量 | 画布显示多条量尺 | 超过上限时提示 |
| 6 | 选中量尺 | 高亮当前量尺 | 可删除或查看详情 | 无 |
| 7 | 删除量尺 | 删除当前量尺 | 画布更新 | 删除失败时保持原状 |
| 8 | 退出量尺 | 关闭量尺模式 | 鼠标恢复默认状态 | 当前量尺是否保留按方案处理 |

## 8. 量尺计算规则

| 项目 | 公式/规则 | 优先级 |
|---|---|---|
| 点坐标 | 使用矩阵坐标，不使用 DOM 像素坐标 | P0 |
| 横向距离 | abs(x2 - x1) x pointWidthDistance | P0 |
| 纵向距离 | abs(y2 - y1) x pointHeightDistance | P0 |
| 直线距离 | sqrt(横向距离² + 纵向距离²) | P1 |
| 单位换算 | mm 转 cm 时除以 10 | P1 |
| 缩放后计算 | 仍按矩阵坐标和点距计算，不按屏幕像素计算 | P0 |
| 翻转后计算 | 翻转前后同两点真实距离一致 | P0 |
| 坐垫/靠背切换 | 使用当前 displayType 对应的点距配置 | P0 |
| 不同设备 | 使用当前设备类型对应点距配置 | P0 |

## 9. 多量尺管理

| 规则 | 说明 | 优先级 |
|---|---|---|
| 数量上限 | 最多 8 条量尺，避免画布混乱 | P1 |
| 量尺颜色 | 默认使用统一颜色，选中时高亮 | P1 |
| 量尺名称 | 可显示“量尺 1、量尺 2” | P2 |
| 选中高亮 | 当前量尺高亮显示 | P1 |
| 单个删除 | 支持删除选中量尺 | P1 |
| 清空全部 | 支持清空所有量尺 | P1 |
| 与框选互斥 | 开启量尺时如已有框选，提示先关闭框选或自动关闭量尺 | P0 |
| 与放大镜关系 | 量尺模式下建议关闭放大镜，防止鼠标交互冲突 | P1 |

## 10. 量尺异常处理

| 异常类型 | 判断条件 | 系统处理 | 用户提示 | 优先级 |
|---|---|---|---|---|
| 非 2D 数字视图 | 当前不是 2D 数字视图 | 不进入量尺模式 | 请在 2D 数字视图下使用量尺 | P0 |
| 与框选冲突 | 框选工具已开启 | 不进入量尺，或提示关闭框选 | 量尺和框选不能同时使用 | P0 |
| 点距缺失 | 当前设备无点距配置 | 不显示真实距离 | 当前设备缺少点距配置，无法计算距离 | P0 |
| 起点越界 | 起点不在有效矩阵内 | 不记录起点 | 请在有效区域内测量 | P0 |
| 终点越界 | 终点不在有效矩阵内 | 不生成量尺 | 请在有效区域内测量 | P0 |
| 起点终点相同 | 两点相同 | 可显示 0，或提示重新选择 | 测量距离为 0 | P1 |
| 量尺过多 | 达到数量上限 | 不新增 | 最多支持 8 条量尺 | P1 |

| 待确认项 | 方案 A | 方案 B | 方案 C | 推荐 |
|---|---|---|---|---|
| 采集中翻转是否允许 | 允许，记录方向变更 | 禁止，避免口径变化 | 弹窗确认 | 方案 B |

## 12. 验收标准

| 编号 | 验收项 | 通过标准 | 优先级 |
|---|---|---|---|
| 验收-翻转-01 | 左右翻转 | 2D 视图按横向镜像显示，rawData 不变 | P0 |
| 验收-翻转-02 | 上下翻转 | 2D 视图按纵向镜像显示，业务矩阵方向同步修正 | P0 |
| 验收-翻转-03 | 90 度旋转 | 支持 0/90/180/270 方向切换，矩阵尺寸和坐标正确更新 | P0 |
| 验收-翻转-04 | 坐垫/靠背独立 | 坐垫和靠背方向配置互不影响 | P0 |
| 验收-翻转-05 | 采集中禁止翻转 | 采集中点击翻转/旋转被禁止并提示 | P0 |
| 验收-翻转-06 | 采集方向 | 采集保存修正方向后的业务矩阵，并记录 data_direction/rotate_degree | P0 |
| 验收-翻转-07 | 方向持久化 | 重新打开软件后恢复上次保存的坐垫/靠背方向 | P0 |
| 验收-翻转-08 | 3D 模型不跟随 | 翻转/旋转不镜像 3D 模型本身 | P1 |
| 验收-量尺-01 | 2D 限制 | 非 2D 数字视图不能使用量尺 | P0 |
| 验收-量尺-02 | 距离计算 | 固定点距下，横向/纵向/直线距离计算正确 | P0 |
| 验收-量尺-03 | 缩放无影响 | 缩放后量尺真实距离不变 | P0 |
| 验收-量尺-04 | 翻转无影响 | 翻转后同两点真实距离不变 | P0 |
| 验收-量尺-05 | 工具互斥 | 框选和量尺不能同时启用 | P0 |
| 验收-量尺-06 | 数量上限 | 最多支持 8 条量尺，第 9 条禁止新增并提示 | P1 |
