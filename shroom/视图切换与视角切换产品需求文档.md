## 2. 视图切换业务逻辑

视图切换只改变用户如何看数据，不改变原始压力矩阵。切换视图时，实时数据流、采集任务、历史数据不应被重建或清空。

| 规则 | 说明 | 优先级 |
|---|---|---|
| 视图状态独立 | 当前视图模式应独立保存，例如 2D 数字、2D 热力图、3D 点图、3D 模型、3D 数字 | P0 |
| 数据源不变 | 视图切换不重新连接设备，不重新采集，不修改 rawData | P0 |
| 工具状态处理 | 切换到不支持某工具的视图时，工具进入暂停/隐藏，不应直接删除用户配置 | P0 |
| 框选处理 | 从 2D 数字切到其他视图时，保留框选区域，但暂停编辑；回到 2D 数字后恢复 | P0 |
| 量尺处理 | 从 2D 数字切到其他视图时，量尺暂停显示或隐藏；回到 2D 数字后按规则恢复或清空 | P1 |
| 放大镜处理 | 仅 2D 数字视图可用，切到其他视图自动关闭 | P1 |
| 缩放状态 | 2D 和 3D 的缩放状态建议分开保存，避免互相污染 | P1 |
| 坐垫/靠背状态 | 座椅类设备切换坐垫/靠背时，工具和框选必须绑定当前 displayType | P0 |
| 采集中切换 | 允许切换视图，但不改变采集数据保存范围 | P0 |
| 回放中切换 | 允许切换视图，当前回放帧不变，只改变渲染方式 | P0 |

## 3. 视图类型规则

| 视图 | 进入条件 | 数据来源 | 展示内容 | 可用工具 | 业务规则 |
|---|---|---|---|---|---|
| 2D 数字视图 | 设备在线并选择 2D 数字 | 实时/历史压力矩阵 | 每个传感点数值 | 框选、量尺、放大镜、翻转、缩放 | 用于精确查看点值 |
| 2D 热力图 | 设备在线并选择热力图 | 实时/历史压力矩阵 | 压力颜色分布 | 翻转、缩放、框选显示 | 用于快速识别压力集中区域 |
| 3D 点图 | 设备在线并选择 3D 点图 | 实时/历史压力矩阵 | 点阵高度和颜色 | 视角、缩放、重置 | 用于观察压力立体分布 |
| 3D 数字 | 设备在线并选择 3D 数字 | 实时/历史压力矩阵 | 3D 数字标签 | 视角、缩放、重置 | 高密度点阵需注意性能 |
| 3D 模型 | 座椅类设备 | 压力矩阵 + 座椅模型 | 座椅模型和压力点 | 视角、整体/坐垫/靠背切换 | 用于演示和空间理解 |

## 4. 视图切换状态保留规则

| 状态/配置 | 切换 2D/3D 是否保留 | 切换坐垫/靠背是否保留 | 说明 |
|---|---|---|---|
| 当前帧数据 | 是 | 是 | 数据源不变 |
| 采集状态 | 是 | 不建议切换系统对象 | 采集中不应影响写库 |
| 回放进度 | 是 | 是 | 切换视图后仍停留当前帧 |
| 缩放比例 | 建议按视图独立保留 | 建议按对象独立保留 | 2D/3D 缩放含义不同 |
| 3D 旋转角度 | 仅 3D 内保留 | 按对象独立保留 | 坐垫和靠背默认角度不同 |
| 框选区域 | 保留但只在 2D 数字可编辑 | 按对象独立保留 | 坐垫框选不能套到靠背 |
| 量尺结果 | 建议只在当前 2D 数字视图保留 | 按对象独立保留 | 防止距离解释错误 |
| 画布翻转 | 保留 | 按对象独立保留 | 坐垫/靠背可有不同方向 |
| 颜色/润滑设置 | 保留 | 保留 | 属于展示参数 |

## 5. 整体/坐垫/靠背切换规则

| 切换动作 | 前置条件 | 系统处理 | 数据影响 | 异常处理 |
|---|---|---|---|---|
| 切换整体 | 座椅类设备 | 显示坐垫和靠背整体 | 不改变原始数据 | 非座椅类隐藏入口 |
| 切换坐垫 | 座椅类设备 | 仅显示坐垫数据 | 不改变原始数据 | 无坐垫数据时提示 |
| 切换靠背 | 座椅类设备 | 仅显示靠背数据 | 不改变原始数据 | 无靠背数据时提示 |
| 坐垫切靠背 | 已有坐垫框选/量尺 | 暂停坐垫工具状态，加载靠背状态 | 不改变数据 | 坐垫区域不得错误套到靠背 |
| 靠背切坐垫 | 已有靠背框选/量尺 | 暂停靠背工具状态，加载坐垫状态 | 不改变数据 | 靠背区域不得错误套到坐垫 |

## 6. 视角切换业务逻辑

视角切换主要用于 3D 点图、3D 数字、3D 模型。它只改变观察角度，不改变压力数据。

| 视角 | 适用视图 | 业务含义 | 系统处理 | 优先级 |
|---|---|---|---|---|
| 默认视角 | 3D 点图、3D 数字、3D 模型 | 打开视图后的标准观察角度 | 进入视图或点击重置时使用 | P0 |
| 正视图 | 3D 模型、靠背 | 从正面观察靠背或整体 | 设置相机位置和目标点 | P1 |
| 俯视图 | 3D 点图、坐垫 | 从上往下看坐垫压力分布 | 设置相机为 top view | P1 |
| 侧视图 | 3D 模型 | 查看压力高度和空间关系 | 设置相机为 side view | P2 |
| 整体视角 | 座椅类 3D 模型 | 同时看到坐垫和靠背 | 设置相机覆盖整体模型 | P0 |
| 坐垫视角 | 座椅类 3D 模型 | 聚焦坐垫区域 | 切换目标对象为坐垫 | P0 |
| 靠背视角 | 座椅类 3D 模型 | 聚焦靠背区域 | 切换目标对象为靠背 | P0 |
| 用户自定义视角 | 3D 视图 | 用户手动旋转后的角度 | 当前会话内保留 | P1 |

## 7. 视角切换规则

| 规则 | 说明 | 优先级 |
|---|---|---|
| 视角切换不影响数据 | 不影响 rawData、displayData、calculatedData、storedData | P0 |
| 视角切换不影响采集 | 采集中切换视角不改变入库内容 | P0 |
| 重置视角 | 恢复当前视图/对象的默认相机位置、缩放、旋转中心 | P0 |
| 旋转中心 | 3D 旋转中心应落在当前对象中心，例如坐垫中心、靠背中心、整体中心 | P0 |
| 对象切换 | 整体/坐垫/靠背切换时，视角和旋转中心同步更新 | P0 |
| 视角记忆 | 同一会话内可记住用户手动调整角度；重新进入软件是否保留待确认 | P1 |
| 视角异常 | 模型加载失败或相机异常时回到默认视角 | P0 |

## 8. 异常处理

| 异常类型 | 判断条件 | 系统处理 | 用户提示 | 优先级 |
|---|---|---|---|---|
| 视图组件加载失败 | 切换后组件未能渲染 | 回退上一个可用视图 | 视图加载失败，已恢复上一视图 | P0 |
| 当前视图不支持工具 | 工具在目标视图不可用 | 工具暂停或隐藏 | 当前视图不支持该工具 | P0 |
| 坐垫/靠背数据缺失 | 当前对象无数据 | 不切换或显示空状态 | 当前无坐垫/靠背数据 | P0 |
| 3D 模型加载失败 | 模型资源缺失或渲染失败 | 回退到 3D 点图或 2D 视图 | 3D 模型加载失败 | P1 |
| 视角重置失败 | 相机状态异常 | 强制恢复默认参数 | 视角已恢复默认 | P1 |
| 采集中切换异常 | 切换视图导致渲染失败 | 不影响采集，回退视图 | 视图切换失败，采集继续 | P0 |

## 9. 待确认项和方案

| 待确认项 | 方案 A | 方案 B | 方案 C | 推荐 |
|---|---|---|---|---|
| 2D/3D 缩放是否共用 | 共用一个缩放值 | 按视图独立保存 | 切换视图时重置 | 方案 B |
| 框选切到非 2D 后怎么处理 | 保留但暂停编辑 | 自动清空 | 弹窗询问 | 方案 A |
| 量尺切到非 2D 后怎么处理 | 暂停显示，回到 2D 恢复 | 自动清空 | 弹窗询问 | 方案 A |
| 3D 视角是否持久化 | 只在当前会话保留 | 写入本地配置长期保留 | 每次进入重置 | V2.0 方案 A |
| 坐垫/靠背是否各自记忆视角 | 分开记忆 | 共用视角 | 每次切换重置 | 方案 A |
| 视图切换时是否允许采集中切换 | 允许，不影响采集 | 禁止切换 | 弹窗确认 | 方案 A |
