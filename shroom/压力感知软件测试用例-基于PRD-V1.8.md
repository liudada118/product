# 压力感知软件测试用例（基于 PRD V1.8）

文档日期：2026-05-11  
产品名称：压力感知软件 JQ Tools  
产品版本：V2.0  
来源文档：`压力感知软件产品需求文档-飞书表格版-含数据处理模块.md` V1.8  
使用技能：`test-scenarios`  
文档用途：给 QA 执行功能测试、回归测试和发布准入验收，可直接复制到飞书表格  

## 1. 测试范围

| 模块 | 测试重点 | 是否发布阻塞 |
|---|---|---|
| 连接流程 | 首次配置、一键连接、重新连接、连接锁、采集前强校验 | 是 |
| 数据处理 | 帧解析、矩阵转换、采样率、坏帧处理、数据层隔离 | 是 |
| 展示视图 | 2D 数字、3D 点图、3D 数字、3D 模型、缩放、颜色上限 | 是 |
| 功能工具 | 置零、翻转、旋转、框选、量尺、放大镜 | 是 |
| 统一分析侧栏 | 框选、历史、对比、导出共用一个侧栏，不允许多个抽屉遮挡 | 是 |
| 数据统计 | 平均、最大、最小、总和、受压面积、重心、图表 30 帧 | 是 |
| 采集回放 | 开始/停止、历史管理、回放、倍速、历史框选、异常恢复 | 是 |
| CSV 与对比 | CSV 导入导出、框选字段、A/B 对比、差值图、对比页侧栏 | 是 |

## 2. 测试准备

| 准备项 | 用途 | 必要性 |
|---|---|---|
| 合法汽车坐垫设备 | 连接、展示、采集、框选、导出 | 必须 |
| 合法汽车靠背设备 | 坐垫/靠背切换、方向独立、对比对象校验 | 必须 |
| 未授权或未配置 MAC 设备 | 非法设备拦截 | 必须 |
| 正常压力帧样例 | 展示、统计、采集、回放 | 必须 |
| 坏帧、断帧、长度不匹配样例 | 异常帧处理 | 必须 |
| 超过 30 帧的历史记录 | 图表缓存和历史回放完整帧 | 必须 |
| 两条同类型历史记录 | A/B 数据对比 | 必须 |
| 正确 CSV、legacy CSV、错误 CSV | CSV 导入导出 | 必须 |
| 小屏或低分辨率环境 | 统一分析侧栏降级测试 | 建议 |
