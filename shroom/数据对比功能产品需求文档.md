# 数据对比功能产品需求文档

文档日期：2026-05-09  
产品名称：压力感知软件 JQ Tools  
模块名称：数据对比  
版本范围：V2.0  
文档用途：给产品、研发、算法、QA 使用，用于明确数据对比功能的业务逻辑、数据口径、交互规则、异常处理和验收标准。  
推荐方案：V2.0 先做“两条历史记录 A/B 对比”，不做实时对比、不做多组对比、不做 3D 对比。

## 1. Summary

数据对比功能用于比较两条历史采集记录，帮助用户判断两个座椅方案、两次调节、两组实验之间的压力分布差异。V2.0 重点实现 A/B 并排图、B-A 差值图、同步帧查看、指标差值和同区域框选对比。

该功能不是简单打开两条历史数据，而是把历史全量矩阵重新计算成“哪里变化、变化多少、是否改善”的分析结果。

## 3. Background

### 3.1 背景

JQ Tools 当前已经支持实时展示、历史数据、数据回放、框选、统计图表、CSV 导入导出。用户完成一次采集后，可以看到单次压力分布，但还不能清楚判断两次数据之间的变化。

在汽车座椅、床垫、工学椅和科研测试场景中，用户经常需要回答这些问题：

| 问题 | 示例 |
|---|---|
| 调整是否有效 | 调整腰托后，腰部压力峰值是否降低？ |
| 方案哪个更好 | 坐垫方案 A 和方案 B 哪个受压更均匀？ |
| 局部变化在哪里 | 坐垫左侧、右侧、靠背腰部区域分别变化多少？ |
| 实验结果能否复核 | 两次实验的差值、变化率和关键指标能否导出？ |

### 3.2 为什么现在做

| 原因 | 说明 |
|---|---|
| 产品闭环需要 | 当前已有采集和回放，缺少“对比分析”会让历史数据价值不足。 |
| 竞品对标需要 | XSENSOR、Tekscan 类产品都强调记录、分析、前后对比和区域分析。 |
| 客户场景需要 | 汽车座椅研发和科研客户更关注“变化”和“改善”，不是只看单次热力图。 |
| 技术基础已有 | 当前已有历史数据、回放、图表、框选和 `/getContrastData` 对比接口线索。 |

### 3.3 当前实现基础

| 当前能力 | 对数据对比的价值 |
|---|---|
| 历史数据列表 | 可作为选择 A/B 记录的入口。 |
| 历史回放 | 可复用播放、暂停、进度条、倍速能力。 |
| 全量矩阵存储 | 可基于原始矩阵重新计算差值和指标。 |
| 框选模块 | 可做同区域局部对比。 |
| 图表模块 | 可扩展 A/B 曲线和差值曲线。 |
| CSV 导出 | P1 可扩展为对比结果导出。 |
| 对比显示模式 | 当前实现文档中已有 `display=contrast` 线索。 |
| 对比接口 | 当前实现文档中已有 `/getContrastData` 线索。 |

### 7.2 主流程

| 步骤 | 用户动作 | 系统处理 | 成功结果 | 异常处理 |
|---|---|---|---|---|
| 1 | 进入历史数据 | 加载历史记录列表 | 显示可选历史数据 | 加载失败提示 |
| 2 | 选择 A 记录 | 记录 A 的基础信息 | A 被选中 | 数据损坏时提示 |
| 3 | 选择 B 记录 | 记录 B 的基础信息 | B 被选中 | 数据损坏时提示 |
| 4 | 点击开始对比 | 校验 A/B 是否可比 | 进入对比模式 | 不可比时阻止进入并说明原因 |
| 5 | 查看 A/B 图 | 加载当前帧矩阵 | A/B 并排显示 | 当前帧缺失时显示空状态 |
| 6 | 查看差值图 | 计算 B-A | 显示差值热力图 | 无法计算时提示 |
| 7 | 拖动进度条 | 按百分比同步 A/B 帧 | 三图同步变化 | 帧数不同按百分比映射 |
| 8 | 查看指标表 | 计算 A、B、差值、变化率 | 指标表更新 | A=0 时变化率显示 N/A |
| 9 | 创建框选 | 同一坐标作用于 A/B | 显示局部对比指标 | 越界时不创建 |
| 10 | 退出对比 | 清理对比状态 | 返回历史数据或主界面 | 不修改历史原始数据 |

### 7.3 页面结构

| 区域 | 内容 | 要求 |
|---|---|---|
| 顶部工具栏 | 选择 A、选择 B、开始对比、退出对比、导出 | 操作路径短，状态清晰。 |
| 左侧视图 | A 数据图 | 显示记录名称、设备类型、时间、当前帧。 |
| 中间视图 | B 数据图 | 显示记录名称、设备类型、时间、当前帧。 |
| 右侧视图 | 差值图 B-A | 使用红蓝差值图，并提供图例。 |
| 下方控件 | 播放、暂停、进度条、倍速 | 复用回放交互，A/B 同步。 |
| 指标面板 | A、B、差值、变化率 | 支持全量和框选区域。 |
| 框选工具 | 同区域框选 | 一个框选同时作用于 A/B 和差值图。 |

### 7.4 对比状态机

| 状态 | 进入条件 | 可执行操作 | 禁止操作 | 退出条件 |
|---|---|---|---|---|
| 未选择 | 进入历史数据 | 选择 A、选择 B | 开始对比 | 选择记录 |
| 已选 A | 用户选择 A | 选择 B、重选 A | 开始对比 | 选择 B |
| 已选 A/B | 用户选择 A 和 B | 开始对比、重选记录 | 无 | 点击开始对比 |
| 校验中 | 点击开始对比 | 等待 | 重复开始 | 校验成功或失败 |
| 对比中 | 校验通过 | 播放、暂停、拖动、框选、查看指标 | 删除正在对比的记录 | 退出对比 |
| 对比异常 | 数据损坏或不可比 | 返回选择、重选数据 | 播放、导出 | 用户处理 |
| 退出对比 | 用户点击退出 | 返回历史或主界面 | 无 | 状态清理完成 |

### 7.5 数据选择规则

| 规则 | 说明 | 优先级 |
|---|---|---|
| 只支持两条历史记录 | V2.0 不做多组对比。 | P0 |
| A 为基准数据 | 差值计算使用 B-A。 | P0 |
| B 为对比数据 | B 通常是调整后、方案 B、实验组。 | P0 |
| 可重选 A/B | 用户进入对比前可以重选。 | P0 |
| 进入对比后可退出重选 | 不在对比页里直接改 A/B，降低状态复杂度。 | P1 |
| 历史记录必须有全量矩阵 | 没有全量矩阵不能参与对比。 | P0 |

### 7.6 数据校验规则

| 校验项 | 通过条件 | 失败提示 | 优先级 |
|---|---|---|---|
| 设备类型 | A/B 设备类型一致 | 两组数据设备类型不同，不能直接对比。 | P0 |
| 矩阵尺寸 | A/B 行列一致 | 两组数据矩阵尺寸不同，不能直接对比。 | P0 |
| 坐垫/靠背 | A/B 当前对象一致 | 请分别选择坐垫对坐垫、靠背对靠背。 | P0 |
| 数据结构 | A/B 都包含全量矩阵 | 数据缺少全量矩阵，不能对比。 | P0 |
| 帧数 | A/B 至少各有 1 帧 | 数据为空，不能对比。 | P0 |
| 压力单位 | A/B 单位一致 | 两组数据单位不一致，不能直接对比。 | P0 |
| 置零口径 | 建议一致 | 两组数据置零口径不同，对比结果仅供参考。 | P1 |
| 翻转方向 | 可统一到显示方向 | 如无法统一，提示用户先调整方向。 | P0 |

### 7.7 帧同步规则

推荐方案：按播放进度百分比对齐。

| 项目 | 规则 |
|---|---|
| 对齐方式 | 以 0%-100% 进度同步 A/B。 |
| 当前帧计算 | A 当前帧 = A 总帧数 x 当前进度；B 当前帧 = B 总帧数 x 当前进度。 |
| 帧数不同 | 允许对比，按百分比映射到最近帧。 |
| 采样率不同 | V2.0 允许对比，但提示“采样率不同，按进度对齐”。 |
| 播放速度 | 播放速度作用于统一进度条。 |
| 拖动进度 | 拖动后 A/B 同步跳到对应百分比帧。 |

备选方案：

| 方案 | 内容 | 优点 | 缺点 | 推荐 |
|---|---|---|---|---|
| 方案 A | 按帧序号对齐 | 简单 | 帧数不同不准确 | 不推荐 |
| 方案 B | 按进度百分比对齐 | 适合不同帧数 | 不是严格时间对齐 | 推荐 V2.0 |
| 方案 C | 按时间戳对齐 | 最严谨 | 依赖时间戳质量 | P1/P2 |

### 7.8 差值计算规则

| 指标 | 公式 | 异常处理 |
|---|---|---|
| 单点差值 | B 点值 - A 点值 | 任一值缺失时显示空。 |
| 指标差值 | B 指标 - A 指标 | 任一指标缺失时显示空。 |
| 变化率 | (B - A) / A x 100% | A=0 时显示 N/A。 |
| 绝对变化量 | abs(B - A) | P1，如需要展示变化强度。 |

### 7.9 差值图规则

| 项目 | 规则 |
|---|---|
| 图类型 | 红蓝差值图。 |
| 颜色含义 | 红色表示 B 大于 A，蓝色表示 B 小于 A，灰色/白色表示接近无变化。 |
| 图例 | 必须显示颜色含义和数值范围。 |
| 默认范围 | 以当前 A/B 差值最大绝对值作为上下限，保证正负对称。 |
| 阈值过滤 | P1 支持隐藏小于阈值的变化。 |
| 交互 | 鼠标悬停可查看 A 值、B 值、差值、变化率。 |

### 7.10 指标对比规则

| 指标 | V2.0 是否做 | 计算范围 | 说明 |
|---|---|---|---|
| 平均压强 | 是 | 全量/框选 | 排除 0。 |
| 最大压强 | 是 | 全量/框选 | 取最大值。 |
| 最小压强 | 是 | 全量/框选 | 排除 0。 |
| 压力总和 | 是 | 全量/框选 | 排除 0。 |
| 受压面积 | 是 | 全量/框选 | 单位 cm²。 |
| 有效受压点数 | 是 | 全量/框选 | 大于阈值的点数。 |
| 压力重心 | 是 | 全量/框选 | 排除 0。 |
| 左右均衡指数 | 否 | 全量/框选 | P1。 |
| 前后均衡指数 | 否 | 全量/框选 | P1。 |
| 正态分布指标 | 否 | 无 | V2.0 不做。 |

### 7.11 框选对比规则

推荐方案：一个框选区域同时作用于 A/B 和差值图。

| 规则 | 说明 | 优先级 |
|---|---|---|
| 同坐标区域 | 用户画一个区域，A/B 使用同一坐标范围。 | P0 |
| 同时作用三图 | 框选区域显示在 A 图、B 图、差值图上。 | P0 |
| 区域指标 | 显示该区域的 A、B、差值、变化率。 | P0 |
| 多框选 | 可复用框选模块，最多 4 个区域。 | P1 |
| 区域命名 | 支持区域命名。 | P1 |
| A/B 独立框选 | V2.0 不做。 | P2 |

### 7.12 数据导出规则
