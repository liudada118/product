# JQ Tools V2.0 压力感知软件 PRD（开发版）

## 1. 文档说明

### 1.1 文档目的

本文档用于给研发、测试、硬件、算法和交付团队明确 JQ Tools V2.0 的开发范围、模块拆分、功能边界、数据要求和验收标准。

本文档后续会继续补充每个功能的详细逻辑，包括按钮状态、数据流、异常处理、字段定义、交互细节和验收用例。

### 1.2 产品定位

JQ Tools V2.0 是面向汽车座椅研发、展厅演示和科研采集的压力感知软件。软件连接适配传感器后，可以实时显示压力数据，支持 2D/3D 可视化、区域分析、采集回放、数据导出和后续数据对比。

### 1.3 V2.0 核心目标

1. 打通从打开软件、连接设备、实时显示、采集、回放到导出的完整闭环。
2. 支持汽车坐垫和汽车靠背同时显示。
3. 支持设备自动识别、MAC 读取、采样率读取和离线授权/固件校验。
4. 支持 2D 热力图、2D 数字矩阵和 3D 点图的基础展示。
5. 支持框选、量尺、可视化调节等分析工具的模块化扩展。
6. 支持实时统计、趋势图、压力重心轨迹等数据可视化能力。
7. 为后续数据导入、数据对比、报告生成和科研分析打基础。

### 1.4 主要用户

| 用户角色 | 使用目的 |
|---|---|
| 测试工程师 | 连接设备、采集压力数据、回放、导出 CSV |
| 展示人员 | 在展厅或门店中实时展示坐垫/靠背压力图 |
| 科研用户 | 查看设备参数、记录实验数据、导出可分析数据 |
| 交付/售后人员 | 安装软件、连接设备、检查授权、排查问题 |
| 研发/QA | 验证协议、数据、性能、异常恢复和功能逻辑 |

## 3. 模块一：打开软件后的连接流程

### 3.1 模块目标

用户打开软件后，可以快速完成设备连接、设备识别、授权校验和设备参数读取。连接成功后，软件进入实时展示状态。

### 3.2 用户流程

1. 用户打开 JQ Tools。
2. 软件进入未连接状态。
3. 用户点击“一键连接”。
4. 软件自动检测串口。
5. 软件识别 CH340 设备。
6. 软件尝试波特率：921600、1000000、3000000。
7. 软件读取设备 MAC 地址。
8. 软件读取设备类型。
9. 软件读取采样率和固件版本。
10. 软件执行本地离线授权或固件校验。
11. 校验通过后，软件进入已连接状态。
12. 软件显示设备参数。
13. 软件加载坐垫/靠背展示视图。

### 3.3 功能清单

| 编号 | 功能 | 优先级 | 说明 |
|---|---|---|---|
| CON-01 | 软件启动 | P0 | Windows 安装后可启动，默认中文 |
| CON-02 | 一键连接 | P0 | 点击后自动完成连接流程 |
| CON-03 | 串口自动检测 | P0 | 自动检测可用串口，优先识别 CH340 |
| CON-04 | 波特率自动尝试 | P0 | 按 921600、1000000、3000000 尝试 |
| CON-05 | MAC 地址读取 | P0 | 读取并显示设备 MAC |
| CON-06 | 设备类型自动识别 | P0 | 由固件或设备信息返回 |
| CON-07 | 采样率读取 | P0 | 采样率由设备返回 |
| CON-08 | 固件版本读取 | P0 | 用于设备参数记录 |
| CON-09 | 本地离线授权/固件校验 | P0 | 无联网使用 |
| CON-10 | 设备参数页 | P0 | 展示设备身份和关键参数 |
| CON-11 | 连接失败提示 | P0 | 显示失败原因和重试入口 |

### 3.4 连接状态

| 状态 | 说明 | 可操作项 |
|---|---|---|
| 未连接 | 软件已打开，但没有设备 | 一键连接 |
| 连接中 | 正在扫描串口或尝试波特率 | 不允许重复点击连接 |
| 已连接 | 设备校验通过 | 进入展示、清零、采集 |
| 连接失败 | 未检测到设备或握手失败 | 查看原因、重试 |
| 授权失败 | 设备未通过校验 | 禁止采集 |
| 设备断开 | 采集或展示过程中设备断开 | 提示异常、保留已有数据 |

### 3.5 设备参数字段

| 字段 | 来源 | 是否必填 | 说明 |
|---|---|---|---|
| MAC 地址 | 设备返回 | 是 | 设备唯一标识 |
| 设备类型 | 设备返回 | 是 | 汽车坐垫、汽车靠背等 |
| 固件版本 | 设备返回 | 是 | 用于兼容性判断 |
| 采样率 | 设备返回 | 是 | 写入历史记录和 CSV |
| 串口号 | 软件检测 | 是 | 用于排查连接问题 |
| 波特率 | 软件探测 | 是 | 记录实际连接波特率 |
| 点阵数量 | 设备返回或配置 | 是 | 用于矩阵解析 |
| 压力范围 | 设备返回或配置 | 待定 | 用于科研参数页 |

### 3.6 异常逻辑

| 异常 | 系统处理 |
|---|---|
| 未插入设备 | 显示“未检测到设备” |
| 多个串口设备 | 只连接协议响应正确的设备 |
| 三个波特率均失败 | 显示“波特率探测失败” |
| MAC 读取失败 | 禁止采集，提示读取失败 |
| 设备类型识别失败 | 禁止采集，提示设备类型未知 |
| 授权失败 | 禁止采集，记录日志 |
| 采集中设备断开 | 提示设备断开，保留已采集缓存 |

### 3.7 验收标准

1. 断网环境下软件可启动。
2. 合法设备可一键连接成功。
3. 非法设备不能采集。
4. MAC 地址、设备类型、采样率、固件版本可显示并记录。
5. 连接失败时有明确中文提示。
6. 设备连接日志可用于排查问题。

## 4. 模块二：连接后的展示视图

### 4.1 模块目标

设备连接成功后，软件应实时展示压力数据。展示视图包括 2D 热力图、2D 数字矩阵和 3D 点图。用户可以对视图进行翻转、缩放、重置和可视化调节。

### 4.2 展示视图结构

连接成功后，主界面建议分为：

1. 顶部状态区：设备状态、MAC、设备类型、采样率、采集状态。
2. 左侧或主区域：坐垫视图。
3. 右侧或主区域：靠背视图。
4. 工具栏：连接、清零、采集、下载、回放、框选、量尺、视图切换。
5. 统计区：核心指标和趋势图。
6. 设置区：可视化调节项。

### 4.3 功能清单

| 编号 | 功能 | 优先级 | 说明 |
|---|---|---|---|
| VIEW-01 | 坐垫/靠背同时显示 | P0 | 汽车坐垫和汽车靠背需要同时展示 |
| VIEW-02 | 2D 热力图 | P0 | 实时显示压力分布 |
| VIEW-03 | 2D 数字矩阵 | P0 | 显示每个传感点数值 |
| VIEW-04 | 3D 点图基础视图 | P0/P1 | 用点阵或高度图展示压力变化 |
| VIEW-05 | 2D/3D/数字视图切换 | P0 | 用户可切换显示模式 |
| VIEW-06 | 左右翻转 | P0 | 只影响显示方向 |
| VIEW-07 | 上下翻转 | P0 | 只影响显示方向 |
| VIEW-08 | 鼠标滚轮缩放 | P0 | 范围 10% 到 1000% |
| VIEW-09 | 视图重置 | P0 | 恢复默认缩放和位置 |
| VIEW-10 | 颜色映射 | P0 | 压力值映射为颜色 |
| VIEW-11 | 视角切换 | P1 | 主要用于 3D 视图 |

### 4.4 2D 热力图逻辑

### 4.8 验收标准

1. 连接成功后能实时显示坐垫和靠背。
2. 2D 热力图能随压力变化实时更新。
3. 2D 数字矩阵可打开和关闭。
4. 3D 点图基础视图可显示压力变化。
5. 翻转、缩放、重置不改变原始数据。
6. 缩放范围限制在 10% 到 1000%。
7. 采集期间视图操作不导致界面卡死。

| 最小压强 | 当前帧最小压力值 | 产品已确认排除 0 值 |
| 压力总和 | 当前帧所有有效压力之和 | 待确认单位 |
| 受压面积 | 超过阈值的受压区域面积 | 阈值待确认 |
| 传感点数 | 传感器点阵数量 | 区分总点数和有效点数 |

### 6.4 框选后的统计口径

| 状态 | 统计范围 |
|---|---|
| 未框选 | 全部传感点 |
| 已框选 | 框选区域内传感点 |
| 坐垫框选 | 坐垫区域内选区 |
| 靠背框选 | 靠背区域内选区 |
| 取消框选 | 恢复全部传感点 |

### 6.5 趋势图表

| 图表 | V2.0 要求 | 说明 |
|---|---|---|
| 压力总和曲线 | 可选 | 随时间显示压力总和 |
| 面积变化曲线 | 可选 | 随时间显示受压面积 |
| 压力正态分布图 | 后续 | 需定义统计对象 |
| 压力重心点轨迹 | 后续 | 需处理抖动 |

后续需补充：

1. 图表是否可拖拽。
2. 图表是否可调整大小。
3. 图表采样频率。
4. 图表显示最近多久的数据。
5. 回放时图表是否同步。

### 6.6 图片和截图数据

后续如果支持压力图截图或图片导出，需要定义：

1. 截图范围：坐垫、靠背、全画布、框选区域。
2. 截图内容：热力图、数字矩阵、统计指标、时间戳。
3. 文件格式：PNG、JPG。
4. 文件命名规则。
5. 是否写入历史记录。

### 6.7 验收标准

1. 指标随压力变化实时更新。
2. 最小压强排除 0 值。
3. 受压面积阈值有明确配置或默认值。
4. 框选后指标只统计框选区域。
5. 取消框选后恢复全局统计。
6. 趋势图数据与实时统计一致。

## 7. 模块五：采集、历史数据与回放

### 7.1 模块目标

用户可以开始采集、停止采集、保存历史数据，并像播放视频一样回放压力变化。

### 7.2 功能清单

|---|---|---|---|

| COL-02 | 停止采集 | P0 | 点击“停止”结束 |
| COL-03 | 按钮状态切换 | P0 | 采集/停止互相切换 |
| COL-04 | 采集时长显示 | P0 | 显示当前采集时长 |
| COL-05 | 本地数据存储 | P0 | 停止后形成历史记录 |
| COL-06 | 本地缓存 | P0 | 采集中持续写入缓存 |
| COL-07 | 异常恢复 | P0 | 强退后恢复未完成采集 |
| COL-08 | 历史数据列表 | P0 | 浏览历史记录 |
| COL-09 | 历史搜索 | P1 | 按名称或设备类型搜索 |
| COL-10 | 重命名与备注 | P1 | 管理历史记录 |
| COL-11 | 数据删除 | P1 | 删除历史记录 |

### 7.3 采集流程

1. 设备已连接。
2. 授权或固件校验通过。
3. 设备类型已识别。
4. 用户点击“采集”。
5. 按钮文字变为“停止”。
6. 软件创建采集缓存。
7. 软件持续写入压力矩阵和统计数据。
8. 用户点击“停止”。
9. 按钮文字变为“采集”。
10. 缓存转为正式历史记录。

### 7.4 采集期间限制

| 操作 | 是否允许 | 说明 |
|---|---|---|
| 切换设备类型 | 不允许 | 防止数据结构变化 |
| 断开设备 | 需确认 | 防止误操作 |
| 视图缩放 | 允许 | 不影响采集 |
| 视图翻转 | 允许 | 不影响原始数据 |
| 框选统计 | 允许 | 不影响全量数据保存 |
| 导出 CSV | 建议不允许 | 采集结束后导出 |

### 7.5 历史数据字段

| 字段 | 说明 |
|---|---|
| 记录 ID | 唯一标识 |
| 记录名称 | 默认自动生成，可修改 |
| 备注 | 用户填写 |
| 开始时间 | 采集开始时间 |
| 结束时间 | 采集停止时间 |
| 采集时长 | 自动计算 |
| 设备 MAC | 设备身份 |
| 设备类型 | 坐垫、靠背等 |
| 固件版本 | 设备返回 |
| 采样率 | 设备返回 |
| 是否清零 | 是/否 |
| 清零时间 | 若有清零则记录 |
| 压力矩阵 | 原始数据 |
| 统计指标 | 可回放计算或存储 |

### 7.6 回放逻辑

| 功能 | 说明 |
|---|---|
| 播放 | 按时间顺序播放历史压力帧 |
| 暂停 | 停留在当前帧 |
| 进度条 | 拖动到指定帧 |
| 倍速 | 调整播放速度，不改变原始时间戳 |
| 统计同步 | 回放画面和统计指标同步 |
| 退出 | 返回历史列表或实时界面 |

### 7.7 异常恢复

| 异常 | 处理 |
|---|---|
| 采集中软件强退 | 下次启动提示恢复或丢弃 |
| 采集中设备断开 | 提示异常，保留已有缓存 |
| 缓存写入失败 | 提示风险，记录日志 |
| 恢复失败 | 保留原缓存，不直接删除 |
| 清空缓存 | 只清临时缓存，不删除正式历史 |

### 7.8 验收标准

1. 采集按钮状态正确切换。
2. 停止后生成历史记录。
3. 历史记录可搜索、重命名、备注、删除。
4. 回放支持播放、暂停、进度条、倍速。
5. 回放热力图和统计指标同步变化。
6. 异常强退后可恢复未完成采集。
7. 清空缓存不删除正式历史数据。

## 9. 全局数据规则

### 9.1 原始数据规则

1. 系统必须保存全量原始压力数据。
2. 框选不得改变默认采集范围。
3. 可视化调节不得修改原始数据。
4. 清零状态必须记录。
5. 导出数据必须能追溯设备、时间、采样率和软件版本。

### 9.2 坐垫/靠背数据规则

1. 坐垫和靠背需要支持同时显示。
2. 坐垫和靠背数据需要分别保存。
3. 坐垫和靠背需要分别支持统计。
4. 坐垫和靠背需要分别支持框选。
5. CSV 中坐垫和靠背压力数组需要分字段保存。

### 9.3 性能规则

1. 60fps 是目标指标，不是硬性阻塞指标。
2. 必须记录实际 FPS。
3. 采集期间视图操作不能导致明显卡死。
4. 30 分钟采集后软件不能明显变慢。
5. 大文件导出时不能导致软件无响应。

## 10. 验收标准

### 10.1 V2.0 版本准入

| 检查项 | 通过标准 |
|---|---|
| 安装包 | Windows 可安装、启动、卸载 |
| 离线启动 | 断网可启动 |
| 设备连接 | 合法设备可一键连接 |
| 授权校验 | 非法设备不能采集 |
| 坐垫/靠背 | 可同时显示 |
| 清零 | 清零影响数据基线，不只是按钮变化 |
| 采集 | 可开始、停止、保存 |
| 回放 | 可播放、暂停、拖动、倍速 |
| 导出 | CSV 可导出并可解析 |
| 异常恢复 | 强退后可恢复缓存 |
| 性能 | 有 FPS 和长时间采集记录 |

### 10.2 阻塞级缺陷

出现以下任一问题，不建议通过 V2.0：

1. 合法设备无法稳定连接。
2. MAC 地址读取错误或缺失。
3. 授权失败设备仍可采集。
4. 设备类型识别失败后仍允许采集。
5. 坐垫/靠背无法同时显示。
6. 采集数据无法保存。
7. 停止采集后历史记录丢失。
8. CSV 导出数据缺失或格式无法解析。
9. 清零逻辑只改变按钮状态，不改变数据基线。
10. 框选导致原始全量数据丢失。
11. 异常退出后缓存损坏且无法恢复。
12. 热力图无法实时反映压力变化。

### 10.3 非阻塞但必须记录

1. 60fps 未完全达到。
2. 高密度数字矩阵显示拥挤。
3. 缩放到 1000% 时操作变慢。
4. 长时间采集文件体积过大。
5. CSV 文件过大导致 Excel 打开慢。
6. 未补齐校准和误差报告。
7. 受压面积阈值暂未最终确认。
