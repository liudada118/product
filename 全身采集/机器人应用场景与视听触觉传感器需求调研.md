# 机器人应用场景与视觉/听觉/触觉传感器需求调研

文档日期：2026-05-22  
公司方向：纤维织物触觉传感器、触觉鞋垫、触觉全身衣、触觉手套、机器人触觉皮肤  
调研问题：现在机器人被应用在哪些场景？哪些场景需要视觉、听觉、触觉传感器？  
使用 Skill：`market-segments` + `value-proposition`

## 1. 核心结论

| 结论 | 说明 |
|---|---|
| 机器人已经真实落地的主战场仍是工业制造和仓储物流 | IFR World Robotics 2025 显示，2024 年全球工业机器人安装量约 54.2 万台；专业服务机器人中，运输与物流是最大应用类别。 |
| 人形机器人正在从演示进入工厂和仓库试点 | GXO 与 Agility Robotics 的 Digit 已进入物流仓库商业部署；BMW、Mercedes-Benz 分别与 Figure、Apptronik 进行汽车制造场景试点。 |
| 视觉是绝大多数机器人场景的基础传感器 | 导航、避障、识别、抓取、质检、巡检、目标定位，都离不开 RGB/RGB-D/双目/激光雷达等视觉或空间感知。 |
| 听觉主要在“人与机器人自然交互”和“异常声音检测”里有价值 | 例如服务机器人、护理机器人、家庭机器人、工厂设备异常检测、语音指令、求救声识别。 |
| 触觉在“接触、抓取、人机协作、安全、柔性物体、足底接地”场景中不可替代 | 视觉能看到物体，但触觉能告诉机器人是否真的接触、用了多大力、是否滑移、是否压到人、是否站稳。 |
| 对你们最有价值的场景不是所有机器人，而是“接触密集型机器人” | 包括人形机器人、协作机器人、仓储搬运、医疗康复、农业采摘、服务照护、人机接触安全、机器人皮肤。 |

一句话判断：

> 视觉让机器人知道“在哪里、是什么”，听觉让机器人知道“人说了什么、环境发生了什么声音”，触觉让机器人知道“是否接触、接触多重、是否安全、是否稳定”。

## 3. 场景与三类传感器需求矩阵

评级说明：高 = 任务成败强依赖；中 = 明显加分或特定任务必需；低 = 可选或只在少数子场景需要。

| 场景 | 视觉 | 听觉 | 触觉 | 触觉为什么重要 |
|---|---:|---:|---:|---|
| 工业制造装配 | 高 | 中 | 高 | 抓取是否到位、插装力、碰撞、人机协作安全、夹持稳定性。 |
| 汽车/电子工厂质检 | 高 | 中 | 中 | 视觉负责缺陷检测；触觉用于接插件、柔性件、装配力和接触确认。 |
| 仓储物流搬运 | 高 | 中 | 高 | 箱体抓取、托举、码放、碰撞、人形机器人足底稳定都需要接触信息。 |
| 人形机器人 | 高 | 高 | 高 | 人形机器人要像人一样走路、抓取、避碰、与人交互，触觉是基础感知层。 |
| 医疗手术机器人 | 高 | 中 | 高 | 组织接触力、器械反馈、误伤风险、精细操作都需要力/触觉。 |
| 康复/护理机器人 | 高 | 高 | 高 | 与人体接触多，需要压力安全、姿态辅助、患者反馈和语音交互。 |
| 农业采摘 | 高 | 低 | 高 | 水果软硬、夹持力、滑移、损伤风险需要触觉；视觉负责识别和定位。 |
| 农业除草/喷洒 | 高 | 低 | 低-中 | 主要依赖视觉识别作物和杂草；触觉多用于底盘接地和工具接触。 |
| 餐饮/酒店服务 | 高 | 高 | 中 | 语音交互重要；触觉用于递物、避碰、托盘稳定、人群安全。 |
| 清洁机器人 | 高 | 中 | 中 | 视觉/激光用于导航；触觉用于碰撞、贴边、地面接触、刷头压力。 |
| 巡检/安防 | 高 | 高 | 中 | 视觉识别异常，听觉识别异响/求救，触觉用于复杂地形接地和碰撞。 |
| 建筑施工 | 高 | 高 | 高 | 搬运、扶持工具、危险环境行走、人与机器协作都需要接触安全。 |
| 家庭机器人 | 高 | 高 | 高 | 家庭任务有大量人、宠物、软物体、易碎物和近距离互动。 |
| 教育/陪伴机器人 | 高 | 高 | 低-中 | 主要是语音和视觉互动；如果涉及拥抱、握手、陪护，触觉价值上升。 |

## 4. 五个重点市场分段

### 4.1 Segment A：仓储物流与制造协作机器人

**场景概况**

这是当前机器人商业化最明确的市场。工业机器人在制造业已经大规模部署；服务机器人中运输与物流是最大应用类别。GXO 与 Agility Robotics 的 Digit 部署说明，人形机器人也在优先进入仓储物流。

**JTBD**

客户希望机器人完成高频、重复、危险、劳累的搬运、分拣、上下料和装配辅助任务，同时不改造太多现有场地。

**传感器需求**

| 模态 | 需求 |
|---|---|
| 视觉 | 识别箱体、料框、货架、托盘、人员、路径和障碍物。 |
| 听觉 | 接收语音指令、识别告警声、设备异常声、人类呼叫。 |
| 触觉 | 抓取压力、箱体滑移、托举稳定、碰撞、人机接触安全、足底接地。 |

**你们的产品机会**

1. 机器人手/臂/躯干贴触觉织物，用于接触安全和箱体搬运。
2. 人形机器人脚底触觉鞋垫，用于站立、转身、搬重物时的重心和接地稳定。
3. 人类工人穿触觉手套/鞋垫采集示教数据，用于训练机器人搬运和协作。

**优先级判断**

这是你们最值得优先关注的场景之一，因为客户预算明确，任务重复，触觉能直接解释“抓稳没有、撞到没有、站稳没有”。

### 4.2 Segment B：人形机器人与通用机器人公司

**场景概况**

人形机器人目前正从实验室进入工厂、仓库和家庭任务试点。BMW 使用 Figure 02 做汽车生产试验；Mercedes-Benz 与 Apptronik 试点 Apollo；GXO 与 Agility Robotics 部署 Digit 做仓库任务。

**JTBD**

客户希望人形机器人在为人设计的环境里完成任务，不需要大规模改造场地，同时能安全地和人类共处。

**传感器需求**

| 模态 | 需求 |
|---|---|
| 视觉 | 场景理解、物体识别、导航、手眼协调、人员检测。 |
| 听觉 | 语音指令、环境声音、人与机器人自然交互。 |
| 触觉 | 手部抓取、足底接地、全身碰撞、安全接触、身体平衡、工具使用。 |

**你们的产品机会**

1. 人形机器人“触觉皮肤”：手、前臂、躯干、腿部、足底。
2. 人类示教套件：人穿手套、鞋垫、全身衣采集同构数据。
3. 机器人执行侧数据：机器人穿同样的触觉织物，形成“人类示教 - 机器人执行”的数据对齐。

**优先级判断**

长期价值最高，但短期商业不确定性也高。适合作为战略客户和联合 PoC，不适合第一天就做完整机器人皮肤平台。

### 4.3 Segment C：医疗康复与照护机器人

**场景概况**

医疗机器人包括手术、康复、医院配送、消毒、远程医疗和护理辅助。这个市场监管强、验证周期长，但触觉价值非常明确。

**JTBD**

客户希望机器人辅助医生和护理人员完成高精度、高重复、接触人体的任务，同时保证安全、可控、可追溯。

**传感器需求**

| 模态 | 需求 |
|---|---|
| 视觉 | 手术视野、人体姿态、康复动作评估、病房导航。 |
| 听觉 | 患者语音、呼叫、异常声音、医护指令。 |
| 触觉 | 人体接触压力、康复辅助力度、护理搬扶安全、手术/器械接触反馈。 |

**你们的产品机会**

1. 康复训练服：记录患者身体接触、压力、姿态与动作质量。
2. 护理机器人安全触觉：扶人、抱扶、转移、递物时避免过压。
3. 医疗床垫/座垫/鞋垫扩展：从压力测量走向康复机器人训练数据。

**优先级判断**

触觉必要性很高，但医疗合规和销售周期长。适合作为中长期垂直方向，不建议作为第一个 MVP 主战场。

### 4.4 Segment D：农业采摘与柔性物体操作

**场景概况**

农业机器人正在用于监测、除草、喷洒、采摘、表型分析。采摘和柔性操作对触觉需求最强。

**JTBD**

客户希望机器人替代人工完成高劳动强度任务，同时减少果蔬损伤、提高采摘成功率和作业稳定性。

**传感器需求**

| 模态 | 需求 |
|---|---|
| 视觉 | 识别果实、枝叶、成熟度、障碍物和路径。 |
| 听觉 | 一般较弱，主要用于人机指令和设备异常。 |
| 触觉 | 判断抓握力、果实软硬、是否滑移、是否压伤、末端执行器接触状态。 |

**你们的产品机会**

1. 采摘机器人夹爪触觉皮肤。
2. 人类采摘工手套数据采集，用于学习合理抓握力。
3. 柔性物体操作 benchmark：水果、布料、软包、食品。

**优先级判断**

技术价值高，但场景分散、户外环境复杂。适合做触觉价值样例，不一定适合作为第一批商业客户。

### 4.5 Segment E：服务、家庭、养老与人机交互机器人

**场景概况**

服务机器人在酒店、餐饮、商场、清洁、导览已有一定落地；家庭和养老机器人仍处早期，但长期空间大。

**JTBD**

客户希望机器人能在有人的环境里自然交互、完成递物、提醒、照护、清洁和陪伴任务，并保持安全。

**传感器需求**

| 模态 | 需求 |
|---|---|
| 视觉 | 人体识别、物体识别、导航避障、表情/动作理解。 |
| 听觉 | 语音对话、呼叫、情绪/语调、异常声音。 |
| 触觉 | 握手、递物、拥抱、搀扶、碰撞检测、人体接触安全。 |

**你们的产品机会**

1. 服务机器人外表面触觉皮肤，用于接触安全。
2. 护理/养老场景全身触觉数据，用于训练“安全接触”。
3. 人穿全身衣采集人机互动数据，形成可训练数据集。

**优先级判断**

概念容易打动人，但短期商业成熟度不如工厂和物流。适合作为展示愿景和投资人叙事。

## 5. 视觉、听觉、触觉分别解决什么问题

### 5.1 视觉：解决“空间和对象”

机器人用视觉回答：

1. 我在哪里？
2. 目标物在哪里？
3. 物体是什么？
4. 人在哪里？
5. 障碍物在哪里？
6. 动作路径是否可行？

典型传感器：RGB 相机、深度相机、双目、鱼眼相机、激光雷达、事件相机、热成像。

### 5.2 听觉：解决“语言和环境声音”

机器人用听觉回答：

1. 人在说什么？
2. 谁在叫我？
3. 是否有求救声、撞击声、设备异常声？
4. 环境是否发生异常？
5. 人的情绪和意图是否变化？

典型传感器：麦克风阵列、声源定位、语音识别、异常声音检测、语音情绪识别。

### 5.3 触觉：解决“接触和物理交互”

机器人用触觉回答：

1. 我真的碰到了吗？
2. 碰到哪里了？
3. 压力有多大？
4. 是否滑了？
5. 是否抓稳？
6. 是否压到人？
7. 是否站稳？
8. 物体是软的、硬的、滑的还是易碎的？

典型传感器：压力阵列、柔性触觉皮肤、力/扭矩传感器、足底压力、指尖触觉、织物触觉、接近 + 触觉融合传感器。

## 6. 哪些场景最需要你们的触觉产品

| 优先级 | 场景 | 推荐产品形态 | 为什么优先 |
|---|---|---|---|
| P0 | 人形机器人数据采集 | 触觉手套 + 鞋垫 + 全身衣 + 机器人皮肤 Lite | 与你们“视觉+触觉同步采集”战略最一致。 |
| P0 | 仓储物流搬运 | 手部/臂部触觉 + 足底触觉 + 视觉同步 | 有明确商业任务，抓取、搬运、站稳都需要触觉。 |
| P0 | 人机接触安全 | 全身衣 + 机器人触觉皮肤 | 视觉无法判断接触压力是否安全，触觉价值直观。 |
| P1 | 工业协作装配 | 手套/末端触觉 + 机器人触觉皮肤 | 插装、装配、协作都需要接触确认和安全。 |
| P1 | 医疗康复/护理 | 全身衣 + 压力垫 + 鞋垫 | 价值高，但合规和销售周期长。 |
| P1 | 农业采摘/柔性物体 | 夹爪触觉 + 人类手套示教 | 触觉价值强，但场景复杂。 |
| P2 | 酒店餐饮服务 | 机器人外表面触觉 + 语音视觉交互 | 可展示，但客户付费深度未必高。 |
| P2 | 家庭陪伴 | 全身/机器人皮肤 | 长期空间大，短期产品化难度高。 |

## 7. 你们应该如何切第一阶段方向

如果目标是服务世界模型、人形机器人和机器人基础模型公司，第一阶段不要做“机器人所有场景大全”。应该聚焦一个可证明触觉价值的组合：

> 第一人称视觉 + 触觉手套 + 触觉鞋垫 + 全身衣 Lite，同步采集人类在真实任务中的接触数据。

推荐 Demo 场景：

| Demo | 使用模态 | 证明点 |
|---|---|---|
| 抓取/递物 | 视觉 + 手部触觉 + 听觉指令 | 视觉看到动作，触觉判断抓稳、滑移、力度。 |
| 搬箱/转身/行走 | 视觉 + 鞋垫触觉 + 全身触觉 | 视觉看到移动，触觉判断重心、足底接触、身体碰撞。 |
| 人机安全接触 | 视觉 + 全身衣 + 机器人触觉皮肤 | 视觉看到靠近，触觉判断接触位置、压力和安全。 |

这个 MVP 能同时连接三个客户故事：

1. 对世界模型公司：视觉之外的物理交互监督信号。
2. 对人形机器人公司：人类示教和机器人执行的触觉数据对齐。
3. 对机器人安全客户：人机接触压力可检测、可回放、可训练。

## 8. 决策建议

### 8.1 不要先做的方向

1. 不要先做完整医疗康复方案。
2. 不要先做完整农业采摘方案。
3. 不要先做家庭陪伴机器人方案。
4. 不要先做所有机器人传感器平台。
5. 不要先做复杂云端大数据系统。

这些方向都有价值，但会让第一阶段失焦。

### 8.2 应该先做的方向

第一阶段建议聚焦：

> 面向人形机器人/世界模型公司的视觉 + 全身触觉同步采集 MVP。

原因：

1. 最能体现你们“全身触觉”的差异化。
2. 能同时覆盖手、足、身体接触三类触觉价值。
3. 能和第一人称视觉数据采集趋势连接。
4. 能为机器人穿戴触觉皮肤做铺垫。
5. Demo 容易讲清楚：人是通过视觉、听觉、触觉理解世界，机器人也需要。

### 8.3 一句话产品方向

> 我们为人形机器人和物理世界模型提供视觉 + 全身触觉同步采集系统，让机器人学习人类在真实世界中的接触、抓取、行走、碰撞和安全交互。
