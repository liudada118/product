# 压力方向识别方案

日期：2026-05-17

## 1. 先明确一个关键点

严格来说，**压力是标量，本身只有大小，没有方向**。

如果你现在的压力传感器只输出一个数值，例如 20 kPa、50 N 或一个 ADC 值，那么它只能说明“压得有多大”，不能单独判断“从哪个方向压过来”。

你真正想测的通常不是“压力方向”，而是以下三类问题之一：

| 你想知道的问题 | 工程上对应的量 | 是否能用单点压力值判断 |
| --- | --- | --- |
| 压力是从左边、右边、前面还是后面斜着来的 | 力的方向 / 力向量 | 不能 |
| 压力落在传感器表面的哪个位置 | 压力中心 / 接触中心 | 不能，至少要多点 |
| 物体有没有横向摩擦、滑动、剪切 | 剪切力方向 | 不能，需要剪切力传感结构 |

所以结论是：**只有压力大小时，不能直接得到方向；要知道方向，必须增加结构设计、多点传感或三轴力测量。**

## 2. 可选技术路线

### 路线 A：四点压力差分，低成本估算方向

这是最适合做第一版样机的方案。

做法是在一个弹性受力结构下面放 4 个压力单元，例如左、右、前、后各一个。上面覆盖一个硅胶半球、硅胶凸台或弹性膜。

当力从正上方压下时，4 个传感单元受力接近一致。

当力从左上方、右上方、前上方或后上方斜着压下时，下面几个传感单元的受力会不一样。通过比较左右、前后的压力差，就可以反推出大致方向。

典型结构：

```text
        外部压力
           ↓
      硅胶半球/弹性帽
   ┌─────────────────┐
   │       ↑ 前       │
   │                 │
   │  左   中心   右  │
   │                 │
   │       ↓ 后       │
   └─────────────────┘
      4 个压力传感单元
```

优点：

- 成本低。
- 可以用你现有的压力传感技术继续做。
- 结构简单，适合快速打样。
- 可以输出大致方向，例如左、右、前、后、左前、右后。

缺点：

- 精度依赖结构和标定。
- 如果接触点本身会移动，方向判断会被干扰。
- 它更适合估算“受力偏向”，不是严格的三维力向量。

适用场景：

- 判断受力方向的大致象限。
- 判断坐垫、靠背、鞋垫、机器人触觉、握持触觉中的压力偏移。
- 对方向精度要求不是特别高的产品原型。

### 路线 B：压力阵列，计算压力中心和分布方向

如果你已经能做柔性压力阵列，可以通过多个点的压力分布来判断压力中心。

计算方式：

```text
总压力 F = Σ Pi * Ai

压力中心 Xc = Σ Xi * Pi * Ai / Σ Pi * Ai
压力中心 Yc = Σ Yi * Pi * Ai / Σ Pi * Ai
```

其中：

- `Pi` 是第 i 个点的压力。
- `Ai` 是第 i 个点的面积。
- `Xi, Yi` 是第 i 个点的位置坐标。
- `Xc, Yc` 是整体压力中心。

如果压力中心持续向左移动，可以判断受力偏向左侧；如果持续向前移动，可以判断受力偏向前侧。

优点：

- 能得到压力热图。
- 能判断接触面积、峰值压力、压力中心。
- 适合座椅、床垫、鞋垫、人体工学产品。

缺点：

- 只能判断压力分布和压力中心，不等于真实力的三维方向。
- 如果需要测横向摩擦方向，仅靠普通压力阵列仍然不够。

适用场景：

- 座椅压力分布。
- 足底压力分析。
- 床垫和坐垫舒适性。
- 接触区域和局部高压点检测。

### 路线 C：三轴力传感器，直接测 Fx、Fy、Fz

如果你的目标是准确知道力的方向，最标准的方法是做三轴力传感。

三轴力分别是：

```text
Fx：左右方向的力
Fy：前后方向的力
Fz：垂直方向的力
```

有了三轴力，就可以计算方向：

```text
力大小 |F| = sqrt(Fx² + Fy² + Fz²)

水平偏转角 α = atan2(sqrt(Fx² + Fy²), Fz)

水平方向角 β = atan2(Fy, Fx)
```

其中：

- `α` 表示力相对垂直方向偏了多少。
- `β` 表示力在水平面上朝哪个方向。

实现方式可以是：

- 应变片式三轴力传感器。
- 电容式三轴触觉传感器。
- 霍尔磁感应 + 弹性体位移结构。
- 光学形变测量。
- 六轴力/力矩传感器。

优点：

- 可以真正输出力的方向。
- 可以区分法向力和剪切力。
- 适合精度要求高的场景。

缺点：

- 结构和算法复杂。
- 成本更高。
- 标定难度更大。

适用场景：

- 机器人触觉。
- 抓取力控制。
- 摩擦、滑移、剪切力检测。
- 精密测试设备。

### 路线 D：曲面压力传感器，根据表面法线估算方向

如果你的传感器贴在一个已知曲面上，例如球面、座椅曲面、人体曲面，可以把每个压力点的方向近似看成该点的法线方向。

计算方式：

```text
单点力向量 Fi = Pi * Ai * ni

总力向量 Ftotal = Σ Fi
```

其中：

- `Pi` 是该点压力。
- `Ai` 是该点面积。
- `ni` 是该点表面法线方向。

这个方案适合已知三维曲面的压力阵列，但它仍然主要测的是法向压力，不能可靠测出切向摩擦力。

## 3. 推荐第一版样机方案

如果你现在只有压力大小，建议不要一开始就做复杂三轴传感器。更现实的第一步是：

**做一个四点压力差分方向传感单元。**

### 3.1 硬件结构

建议结构：

1. 底部做一块硬质 PCB 或结构板。
2. 上面放 4 个压力传感单元，位置分别为左、右、前、后。
3. 传感单元上方放一个整体硅胶弹性帽。
4. 硅胶帽顶部作为外部接触点。
5. 每个传感单元单独采集 ADC 数据。

推荐布局：

```text
           前 F
            ●

左 L   ●    +    ●   右 R

            ●
           后 B
```

如果空间允许，可以增加中间一个压力单元：

```text
           前 F
            ●

左 L   ●    ● C    ●   右 R

            ●
           后 B
```

中间点 `C` 用来增强垂直压力测量，四周点用来判断方向。

### 3.2 基础算法

先对每个通道做零点扣除：

```text
L = raw_L - zero_L
R = raw_R - zero_R
F = raw_F - zero_F
B = raw_B - zero_B
```

计算总压力：

```text
S = L + R + F + B
```

计算左右方向偏移：

```text
Dx = (R - L) / S
```

计算前后方向偏移：

```text
Dy = (F - B) / S
```

计算方向角：

```text
方向角 β = atan2(Dy, Dx)
```

计算倾斜程度：

```text
倾斜强度 T = sqrt(Dx² + Dy²)
```

方向判断可以先用简单规则：

| 条件 | 判断结果 |
| --- | --- |
| `S` 很小 | 无有效压力 |
| `T` 很小 | 接近垂直按压 |
| `Dx > 0` | 偏右 |
| `Dx < 0` | 偏左 |
| `Dy > 0` | 偏前 |
| `Dy < 0` | 偏后 |

如果需要输出 8 个方向：

```text
β 在 -22.5° 到 22.5°：右
β 在 22.5° 到 67.5°：右前
β 在 67.5° 到 112.5°：前
β 在 112.5° 到 157.5°：左前
β 在 157.5° 到 180° 或 -180° 到 -157.5°：左
β 在 -157.5° 到 -112.5°：左后
β 在 -112.5° 到 -67.5°：后
β 在 -67.5° 到 -22.5°：右后
```

### 3.3 注意零除和低压力误判

当总压力 `S` 很小时，方向计算没有意义。

建议加阈值：

```text
if S < pressure_threshold:
    状态 = 无压力
else:
    计算方向
```

例如：

```text
pressure_threshold = 空载噪声均值 + 5 * 空载噪声标准差
```

### 3.4 滤波建议

实际信号会抖动，建议做简单滤波：

```text
filtered = 0.8 * old_value + 0.2 * new_value
```

或者使用 5-10 个采样点的滑动平均。

采样频率建议：

| 场景 | 采样频率 |
| --- | --- |
| 静态按压方向 | 20-50 Hz |
| 人体坐姿、靠背、鞋垫 | 50-100 Hz |
| 机器人触觉、抓取 | 100-500 Hz |
| 冲击或高速动态 | 1000 Hz 以上 |

## 4. 标定流程

方向识别能不能用，关键不在公式，而在标定。

建议做一个简单标定治具：

1. 用固定砝码或推拉力计提供已知力。
2. 用角度架提供已知方向，例如 0°、15°、30°、45°。
3. 分别从前、后、左、右、左前、右前、左后、右后方向加载。
4. 每个方向采集 5-10 组数据。
5. 记录每个通道的 ADC 值、真实力大小、真实加载角度。

### 4.1 最小标定表

| 加载方向 | 加载角度 | 力大小 | L | R | F | B | 备注 |
| --- | --- | --- | --- | --- | --- | --- | --- |
| 垂直 | 0° | 10 N |  |  |  |  |  |
| 右偏 | 15° | 10 N |  |  |  |  |  |
| 右偏 | 30° | 10 N |  |  |  |  |  |
| 前偏 | 15° | 10 N |  |  |  |  |  |
| 前偏 | 30° | 10 N |  |  |  |  |  |
| 左前 | 30° | 10 N |  |  |  |  |  |
| 右后 | 30° | 10 N |  |  |  |  |  |

### 4.2 用标定修正方向

最简单的修正方式是给 `Dx`、`Dy` 加比例系数：

```text
Dx_corrected = kx * Dx
Dy_corrected = ky * Dy
β = atan2(Dy_corrected, Dx_corrected)
```

如果结构非线性明显，可以用多项式或机器学习模型：

```text
输入：[L, R, F, B, S, Dx, Dy]
输出：[方向角 β, 倾斜角 α, 总力 Fz]
```

第一版建议先用线性模型，不要一上来就做复杂神经网络。

## 5. 你需要区分的三个结果

### 5.1 压力中心

压力中心表示压力落在哪里。

它可以回答：

- 压力偏左还是偏右？
- 压力集中在哪里？
- 压力中心是否在移动？

但它不一定等于力的方向。

### 5.2 法向压力

普通压力传感器主要测的是垂直于传感器表面的压力。

它可以回答：

- 压得多重？
- 哪个位置压力最大？
- 接触面积多大？

但它不能回答：

- 有没有横向摩擦？
- 物体是不是在滑动？
- 力到底从哪个角度斜着来？

### 5.3 剪切力

剪切力表示沿着传感器表面方向的力。

如果你要检测滑动、摩擦方向、抓取稳定性，就必须加入剪切力检测结构。

## 6. 产品化建议

### 6.1 如果目标是“能判断大概方向”

推荐：

- 四点压力差分。
- 硅胶弹性帽。
- 输出 8 个方向 + 垂直按压 + 无压力。
- 做基本标定。

这是最适合快速落地的方案。

### 6.2 如果目标是“能做压力分布”

推荐：

- 做二维压力阵列。
- 输出压力热图、接触面积、峰值压力、压力中心轨迹。
- 用压力中心变化判断受力偏移。

这适合座椅、床垫、鞋垫、人体工学和汽车舒适性测试。

### 6.3 如果目标是“准确测力的方向”

推荐：

- 做三轴力传感器。
- 直接输出 `Fx、Fy、Fz`。
- 计算力向量、倾斜角和方向角。
- 用六轴力传感器做标定基准。

这适合机器人、抓取、摩擦、滑移检测和高精度测试。

## 7. 第一阶段落地清单

建议第一阶段按下面顺序做：

1. 确定要判断几个方向：4 方向、8 方向，还是连续角度。
2. 做 4 点或 5 点压力传感结构。
3. 设计硅胶弹性帽，保证斜向压力能传递成差分压力。
4. 每个通道独立采集 ADC。
5. 做零点校准和压力阈值判断。
6. 用 `Dx = (R - L) / S`、`Dy = (F - B) / S` 做初版方向算法。
7. 用已知方向和已知力做标定。
8. 验证方向准确率、重复性、迟滞、漂移和温度影响。
9. 再决定是否升级到压力阵列或三轴力传感器。

## 8. 判断方案是否成功的指标

建议至少验证这些指标：

| 指标 | 建议目标 |
| --- | --- |
| 无压力误触发率 | 越低越好，先控制在 1% 以下 |
| 方向分类准确率 | 4 方向至少 90%，8 方向至少 80% |
| 垂直按压误判为斜向 | 尽量低于 5% |
| 重复加载一致性 | 同一方向输出角度波动小于 10° |
| 响应时间 | 人体交互小于 100 ms，机器人触觉按需求提高 |
| 温漂 | 温度变化后需要可补偿 |
| 迟滞 | 加载和卸载曲线差异要记录 |

## 9. 主要风险

1. **单点压力值无法判断方向**  
   如果硬件仍然只有一个压力数值，算法无法凭空算出方向。

2. **压力中心不等于力方向**  
   多点压力可以判断受力位置偏移，但如果接触点本身移动，可能会误认为方向变化。

3. **硅胶结构影响很大**  
   弹性体硬度、厚度、形状、装配预压都会影响方向判断。

4. **剪切力不能靠普通压力片直接测**  
   如果你要判断摩擦方向或滑动方向，需要专门的剪切力结构。

5. **必须标定**  
   不做标定，只靠理论公式，很难稳定量产。

## 10. 推荐结论

如果你现在的产品只有压力大小，想尽快做出“方向感知”，建议从下面方案开始：

**4 个压力传感单元 + 一个硅胶弹性帽 + 差分算法 + 标定。**

第一版目标不要定成“精确三维力方向”，而是定成：

> 能稳定判断无压力、垂直按压、左、右、前、后、左前、右前、左后、右后。

等这个方案跑通后，再根据客户需求升级：

- 要压力分布：做压力阵列。
- 要真实力方向：做三轴力传感。
- 要滑动和摩擦方向：做剪切力传感。
