# 老年人评估系统定位与边界调研

> 版本：v0.1  
> 日期：2026-05-17  
> 调研方法：使用 `customer-research` 梳理目标用户、场景、JTBD 和痛点；使用 `value-proposition` 收敛产品价值主张；结合老年综合评估、ICOPE、肌少症、跌倒风险、吞咽、二便、认知等公开资料判断能力边界。  
> 当前系统能力：手套传感器检测握力；步道/坐垫传感器检测起坐能力、静态站立、行走。

## 1. 一句话结论

你现在解决的不是“全部老年综合评估”问题，而是：

> 老年人身体功能下降的客观化评估问题，重点覆盖肌力、下肢起坐能力、静态平衡、步行能力、肌少症风险、衰弱风险和跌倒风险。

更准确的产品定位是：

> 老年身体功能与跌倒/肌少症风险评估系统。

不建议第一版定位为：

- 老年综合评估系统
- 全面养老评估系统
- 认知/吞咽/二便综合诊断系统
- 老年疾病诊断系统

因为你的四项传感评估主要覆盖“运动功能”和“肌力功能”，不能直接判断吞咽障碍、大小便失禁、认知障碍、营养不良、抑郁、视听功能、用药风险和社会支持。

## 2. 你实际解决的问题

### 2.1 核心问题

传统老年身体功能评估常依赖人工秒表、肉眼观察和手工记录，存在三个痛点：

1. **不够客观**：同一个起坐、站立、行走动作，不同评估人员判断可能不同。
2. **不够细**：人工通常只记录“几秒完成”，看不到过程中的发力、稳定性、左右差异和波动。
3. **不好追踪**：复测时很难判断老人是明显改善、轻微恶化，还是只是测量误差。

你的系统用传感器把“肌力、起坐、站立、行走”变成可量化数据，解决的是：

> 把老年人的运动功能衰退从主观观察，变成可记录、可比较、可追踪的客观评估。

### 2.2 对应老年综合评估中的哪个模块

老年综合评估 CGA 通常包含医学、功能、认知、心理、营养、社会支持、环境、用药、跌倒、二便等多个维度。WHO ICOPE 也强调老年人的内在能力包括运动、认知、活力/营养、感官、心理等维度。

你的系统主要落在这几个模块：

| 老年评估模块 | 当前系统覆盖程度 | 说明 |
| --- | --- | --- |
| 肌力 | 高 | 手套握力可直接量化上肢肌力，是肌少症筛查关键指标之一 |
| 下肢功能 | 高 | 坐垫/步道可评估起坐能力，接近 5 次坐站等标准测试 |
| 平衡能力 | 中高 | 静态站立可反映稳定性、重心波动、左右负重 |
| 步行能力 | 高 | 步道可评估步速、步态周期、步长、步态稳定性 |
| 跌倒风险 | 中高 | 起坐、站立、行走与跌倒风险高度相关，但仍需结合跌倒史、用药、视力、环境等因素 |
| 肌少症风险 | 中高 | 覆盖低肌力和身体功能，但若没有 BIA/DXA/小腿围，不能完整确认肌量下降 |
| 衰弱风险 | 中 | 握力和步速是衰弱相关指标，但还缺疲劳、体重下降、活动量等 |
| 康复效果追踪 | 高 | 适合做干预前后对比和趋势追踪 |

## 3. 四个评估分别解决什么

### 3.1 手套传感器检测握力

解决的问题：

- 老人上肢肌力是否下降。
- 是否存在肌少症风险中的“低肌力”信号。
- 康复、营养、运动干预后握力是否改善。

产品价值：

- 比普通握力计更容易做连续曲线、左右手对比、发力过程分析。
- 可记录最大握力、平均握力、发力速度、持续稳定性、左右差异。

不能解决：

- 不能直接判断肌量是否下降。
- 不能判断认知、吞咽、二便、营养问题。
- 不能解释握力下降的病因，如神经损伤、关节疼痛、帕金森、脑卒中后遗症等。

### 3.2 坐垫/步道检测起坐能力

解决的问题：

- 老人从坐到站是否困难。
- 下肢力量、起立控制和转移能力是否下降。
- 起坐动作是否存在左右受力不均、起立慢、坐下控制差。

产品价值：

- 比人工 5 次坐站只记录总时间更细。
- 可分析每次起立/坐下的时间、峰值压力、左右差异、稳定性变化。

不能解决：

- 不能单独判断老人为什么起不来，可能是肌力、疼痛、关节、神经、低血压、恐惧跌倒等多因素。
- 不能替代 ADL 评估中的洗澡、穿衣、如厕、进食等生活能力评估。

### 3.3 静态站立检测

解决的问题：

- 老人站立稳定性如何。
- 是否存在明显摇晃、左右负重不均、重心控制差。
- 是否适合进入更复杂的步行或 TUG 测试。

产品价值：

- 可作为跌倒风险和身体控制能力的基础指标。
- 可用于康复训练前后对比。

不能解决：

- 不能完整判断眩晕、前庭功能、视力、药物副作用导致的跌倒风险。
- 不能判断认知障碍导致的迷路、判断力下降或安全意识下降。

### 3.4 行走检测

解决的问题：

- 老人走得快不快、稳不稳、步态是否异常。
- 是否存在步速下降、步幅变小、左右不对称、步态周期不稳定。
- 是否有肌少症、衰弱、跌倒风险的功能表现。

产品价值：

- 步速是老年功能状态的重要指标。
- 步道比人工秒表更能记录步态细节。
- 适合做复测趋势和康复效果追踪。

不能解决：

- 不能单独诊断帕金森、脑卒中、周围神经病变、骨关节炎等疾病。
- 不能替代医生的神经系统、骨科、心肺和用药评估。

## 4. 价值主张

### 4.1 Who：给谁用

第一优先级用户：

- 养老机构健康管理人员
- 康复科/康复中心治疗师
- 社区卫生服务中心老年健康管理人员
- 体检中心老年专项评估团队

第二优先级用户：

- 老年医学科
- 跌倒门诊
- 营养门诊
- 慢病管理机构

### 4.2 Why：他们为什么需要

他们需要快速判断：

- 这个老人是不是肌力下降。
- 起坐、站立、行走有没有明显功能下降。
- 是否存在跌倒风险。
- 是否需要进一步做肌少症、衰弱、康复、营养或老年医学评估。
- 干预后有没有改善。

### 4.3 What Before：现在怎么做

当前常见做法：

- 握力计测握力。
- 秒表测 5 次坐站、步速、TUG。
- 评估人员肉眼判断站立和步态。
- 纸质表格或 Excel 记录。
- 报告靠人工整理。

痛点：

- 记录不标准。
- 过程数据缺失。
- 多次复测难比较。
- 家属不容易理解。
- 机构很难批量管理高风险老人。

### 4.4 How：你的系统如何解决

系统通过四类传感评估提供价值：

- 手套握力：量化肌力和左右差异。
- 起坐检测：量化下肢起立能力和转移能力。
- 静态站立：量化平衡和稳定性。
- 行走检测：量化步速、步态和行走稳定性。

最终输出：

- 身体功能评分
- 肌少症风险提示
- 跌倒风险提示
- 起坐/站立/行走专项报告
- 复测趋势
- 建议进一步评估的方向

### 4.5 What After：用了以后变成什么样

机构可以从“凭经验看老人走得稳不稳”，变成：

- 有标准动作流程。
- 有客观指标。
- 有风险分层。
- 有复测趋势。
- 有家属能看懂的报告。
- 有转诊或干预依据。

### 4.6 Alternatives：替代方案是什么

| 替代方案 | 优点 | 不足 |
| --- | --- | --- |
| 普通握力计 + 秒表 | 成本低、易普及 | 缺过程数据，难做趋势和自动报告 |
| 人工 CGA 量表 | 全面 | 耗时、依赖专业人员，运动功能不够细 |
| BIA 体成分 | 可估算肌量 | 不评估起坐、站立、行走 |
| DXA | 肌量参考价值高 | 昂贵，不适合养老/社区批量筛查 |
| 步态分析设备 | 步态参数丰富 | 通常不覆盖握力和起坐，成本较高 |
| 跌倒风险量表 | 快速 | 主观性强，不能量化动作过程 |

## 5. 你解决不了的问题

### 5.1 吞咽问题

吞咽障碍关系到误吸、肺炎、营养不良和进食安全，通常需要 EAT-10、饮水试验、言语治疗师评估，必要时还需要 FEES 或 VFSS 等专业检查。

当前系统不能解决：

- 是否存在吞咽障碍。
- 是否有误吸风险。
- 是否需要糊状饮食、增稠液体或吞咽训练。
- 是否需要言语治疗师介入。

可做的产品延伸：

- 加 EAT-10 问卷。
- 加“饮水呛咳、进食时间变长、体重下降、反复肺炎”风险提示。
- 输出“建议吞咽专科/言语治疗师进一步评估”，不要自动判断吞咽诊断。

### 5.2 大小便问题

尿失禁、便秘、便失禁、夜尿和如厕能力属于老年综合评估、护理等级和长期照护中的重要问题。

当前系统不能解决：

- 尿失禁类型和严重程度。
- 便秘或便失禁原因。
- 夜尿、如厕转移、护理依赖。
- 是否需要泌尿、消化、盆底康复或护理干预。

可做的产品延伸：

- 加 ICIQ-UI SF 等尿失禁简表。
- 加 Bristol 大便性状、排便频率、便秘风险问题。
- 加如厕转移能力和护理依赖记录。

### 5.3 认知问题

认知评估需要看记忆、定向、注意、执行功能、语言、视空间等能力。Mini-Cog、MMSE、MoCA 常用于筛查，但筛查结果也不能直接诊断痴呆。

当前系统不能解决：

- 是否有轻度认知障碍或痴呆。
- 老人是否能独立服药、理财、做饭、外出。
- 跌倒是否与认知、判断力或注意力下降有关。
- 是否有谵妄、抑郁导致的认知表现变化。

可做的产品延伸：

- 加 Mini-Cog 作为快速筛查。
- 医疗场景可接 MMSE/MoCA 录入。
- 加 IADL 问卷判断复杂生活能力。
- 报告中把“身体功能风险”和“认知风险”分开。

### 5.4 营养问题

肌少症和营养密切相关，但你的系统目前主要测功能，不直接评估营养摄入和营养不良。

当前系统不能解决：

- 蛋白质摄入是否不足。
- 是否存在营养不良风险。
- 体重下降是否由进食不足、吞咽问题、疾病或抑郁引起。
- 是否需要营养处方。

可做的产品延伸：

- 加 MNA-SF 或 NRS 2002。
- 加体重变化、食欲、进食量、蛋白摄入、BMI。
- 接入 BIA 或体成分数据。

### 5.5 心理、情绪和睡眠问题

抑郁、焦虑、睡眠差会影响活动能力、食欲、跌倒风险和康复依从性。

当前系统不能解决：

- 是否抑郁或焦虑。
- 是否有睡眠障碍。
- 活动减少是身体功能下降，还是情绪低落导致。

可做的产品延伸：

- 加 GDS-15、PHQ-9 或简短睡眠问题。
- 报告中提示“建议心理/精神专科进一步评估”。

### 5.6 视力、听力、用药和环境问题

跌倒风险不只来自肌力和平衡，也来自视力、听力、药物、低血压、家居环境等。

当前系统不能解决：

- 视力/听力下降导致的风险。
- 多重用药、镇静药、降压药导致的跌倒风险。
- 家中地毯、台阶、照明、卫生间扶手等环境风险。
- 骨质疏松和骨折风险。

可做的产品延伸：

- 加用药清单和跌倒相关药物提示。
- 加视力/听力筛查问题。
- 加家庭环境风险清单。
- 加骨质疏松筛查或 DXA 结果录入。

## 6. 产品边界图

| 问题类别 | 当前能解决吗 | 适合的表达 |
| --- | --- | --- |
| 握力下降 | 能 | 客观评估肌力 |
| 起坐困难 | 能 | 评估下肢功能和转移能力 |
| 站立不稳 | 部分能 | 评估静态平衡风险 |
| 走路慢/步态不稳 | 能 | 评估步行能力和跌倒相关风险 |
| 肌少症风险 | 部分能 | 识别低肌力和低身体功能，建议进一步肌量评估 |

| 跌倒风险 | 部分能 | 评估运动功能相关跌倒风险，不覆盖所有原因 |
| 康复效果追踪 | 能 | 训练前后客观对比 |
| 吞咽障碍 | 不能 | 只能加问卷筛查和转介建议 |
| 大小便问题 | 不能 | 只能加问卷和护理记录 |
| 认知障碍 | 不能 | 只能加 Mini-Cog/MMSE/MoCA 录入 |
| 营养不良 | 不能 | 需要 MNA-SF、体重、饮食、BIA 等 |
| 抑郁焦虑 | 不能 | 需要 GDS/PHQ 等心理筛查 |
| 用药风险 | 不能 | 需要药师或医生审查 |
| 社会支持/环境 | 不能 | 需要访谈和环境评估 |

## 7. 推荐产品定位

### 7.1 最清晰的定位

> 一套面向养老、社区、康复和体检场景的老年身体功能评估系统，通过握力、起坐、站立和行走四项传感检测，客观识别肌力下降、下肢功能下降、平衡异常、步行能力下降和跌倒/肌少症风险，并支持复测追踪。

### 7.2 对外销售话术

机构版：

> 帮助机构用 10-15 分钟完成老年人身体功能评估，自动生成握力、起坐、站立、行走和风险趋势报告，快速识别需要重点关注、康复干预或进一步医学评估的人群。

医生/康复师版：

> 把传统依赖秒表和肉眼观察的老年功能评估，变成可量化、可复测、可追踪的客观数据，辅助判断肌少症、衰弱和跌倒风险。

家属版：

> 看清老人力量、起立、站稳和走路能力有没有下降，知道是否需要锻炼、康复或进一步就医。

### 7.3 不建议的宣传话术

- 全面评估所有老年健康问题。
- 自动诊断肌少症。
- 自动预测所有跌倒。
- 可检测认知障碍、吞咽障碍、大小便异常。
- 替代医生、康复师或老年综合评估。

## 8. 推荐产品路线

### 8.1 V0.1：身体功能专项评估

聚焦你现有四项能力：

- 握力
- 起坐
- 静态站立
- 行走
- 风险报告
- 复测趋势

目标：证明传感器比人工秒表和普通表格更有价值。

### 8.2 V0.2：肌少症/衰弱风险增强

补充低成本问卷和测量：

- SARC-F/SARC-CalF
- 小腿围
- 体重下降
- 疲劳感
- 活动量下降
- 跌倒史
- BIA/DXA 结果录入

目标：从“身体功能评估”升级为“肌少症/衰弱风险评估”。

### 8.3 V0.3：老年综合筛查入口

不自己诊断所有问题，而是加入筛查和转介：

- Mini-Cog：认知筛查
- EAT-10：吞咽风险筛查
- ICIQ-UI SF：尿失禁筛查
- MNA-SF：营养风险筛查
- GDS-15：抑郁筛查
- ADL/IADL：生活能力筛查

目标：把系统变成“身体功能评估为核心的老年综合筛查入口”。

### 8.4 V1.0：机构管理平台

面向养老、社区、体检和康复机构：

- 批量老人档案
- 高风险人群列表
- 复测提醒
- 干预记录
- 家属报告
- 机构统计看板
- 数据导出
- 权限和隐私管理

## 9. 建议先验证的关键假设

1. 机构是否认为四项传感评估比普通握力计 + 秒表更有价值。
2. 老人是否能在 10-15 分钟内完成四项评估。
3. 评估人员是否能无医生在场独立操作。
4. 报告是否能帮助判断谁需要康复、运动、营养或进一步医学评估。
5. 家属是否愿意为“看得懂的功能趋势报告”付费。
6. 传感指标是否能稳定复测。
7. 握力、起坐、站立、行走指标是否与 SARC-F、跌倒史、SPPB、BIA 等结果相关。
